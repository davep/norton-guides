 How do I print USA postal barcodes with my CA-Clipper application?

 Try the C Gazette issue of Winter 1990-91 (Volume 5, Number 2) for
 details on the format.  If you know C the translation into
 CA-Clipper can't be very hard.  If you don't know C you will find
 the syntax embarrassingly similar to CA-Clipper anyway.
