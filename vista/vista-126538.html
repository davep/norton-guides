   D_RECOVER

SUMMARY
   d_recover(logfile)
   char *logfile;

ARGUMENTS
   logfile      Name of transaction log file to be recovered.

DESCRIPTION
   This function is called by the db_VISTA runtime system to perform
   autorecovery of a failed process transaction. The name of the transaction
   log file to be recovered is contained in the string logfile.

──────────────────────────────────────────────────────────────────────────────
NOTE
   This function should never need to be used by an application program.

──────────────────────────────────────────────────────────────────────────────

CURRENCY CHANGES
   None;

RETURN CODES

   No special codes
