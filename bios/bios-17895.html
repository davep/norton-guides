0040h:0008h           Parallel Printer Port Base Addresses

    Length: 8 bytes

    Base addresses for four parallel printer ports. If a port address
    is 0, then that port is not used.

                0040h:0008h - Parallel Printer Port #1 (LPT1)
                0040h:000Ah - Parallel Printer Port #2 (LPT2)
                0040h:000Ch - Parallel Printer Port #3 (LPT3)
                0040h:000Eh - Parallel Printer Port #4 (LPT4)
