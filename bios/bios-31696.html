0040h:0065h           Current Setting of the Mode Control Register

    Length: 1 byte

    Contains the current setting of the Mode Control register (port
    03x8h).

    ──────────────────────────────────────────────────────────────────

    Depending upon which CRT controller is currently active, this
    value could reflect either the contents of:

                03B8h for Monochrome, or
                03D8h for Color Graphics Adapter
