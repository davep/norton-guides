0040h:004Eh           Starting Address of Regeneration Buffer

    Length: 2 bytes

    This value is the offset from the beginning of the display adapter
    memory. It is usually 0, since the first display page is most
    often used. However, if the display page is changed to 2, this
    value is set equal to twice the value at 0040h:004Ch to reflect
    the new display page. Thus the value at this location is always a
    multiple of the word at 0040h:004Ch.
