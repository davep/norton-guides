0040h:001Ah           Keyboard Buffer Head Pointer

    Length: 2 bytes

    This word points to the next character code/scan code to be
    removed from the keyboard buffer. If the head pointer is equal to
    the tail pointer (0040h:001Ch), then the buffer is empty.
