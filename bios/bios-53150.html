FFFFh:0005h           ROM BIOS Release Date, in ASCII

    Length: 8 bytes

    Contains the ROM BIOS Release Date in ASCII (e.g., '06/10/85').

    Later versions of the Compaq have this release date shifted by 1
    byte, to start at FFFFh:0006h.
