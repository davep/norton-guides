Int 10h, 1123h          Load ROM 8x8 Graphics Font             EGA, MCGA, VGA

    Loads ROM BIOS 8x8 character font in graphics mode.

        Entry   AX = 1123h
                BL = Number of rows
                     | 00h - DL contains number of rows
                     | 01h - 14 (0Eh) rows
                     | 02h - 25 (19h) rows
                     | 03h - 43 (2Bh) rows

        Return  Nothing
