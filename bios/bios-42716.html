0040h:008Eh           Fixed Disk Interrupt Control

    Length: 1 byte

    This byte is reserved for the PC, PCjr, PC XT dated 11/8/82, and
    the PC Convertible. For all others, this byte contains the
    interrupt flag for the fixed disk.
