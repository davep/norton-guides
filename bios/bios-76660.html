Int 10h, 1102h          Load ROM BIOS 8x8 Alpha Font           EGA, MCGA, VGA

    Changes font to ROM BIOS 8x8 font (double-dot set) in text mode.

        Entry   AX = 1102h
                BL = Font block to load (EGA/MCGA 0..3 | VGA 0..7)

        Return  Nothing
