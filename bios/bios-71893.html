Int 10h, 1010h          Update DAC Register                         MCGA, VGA

    Sets the value of one video DAC register.

        Entry   AX = 1010h
                BX = Color register number
                DH = Red color value     \
                CH = Green color value    > 6-bit values
                CL = Blue color value    /

        Return  Nothing
