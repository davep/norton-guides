Int 10h, 1013h          Set DAC Color Page                                VGA

    Selects DAC color paging mode.

        Entry   AX = 1013h
                BL = 00h
                BH = | 00h    4 pages of 64 registers
                     | 01h   16 pages of 16 registers

        Return  Nothing

    ──────────────────────────────────────────────────────────────────

    Sets the active color page.

        Entry   AX = 1013h
                BL = 01h
                BH = Page number

        Return  Nothing
