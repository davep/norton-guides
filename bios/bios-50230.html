0040h:00F0h           Intra-Application Communications Area (ICA)

    Length: 16 bytes

    These sixteen bytes, called the Intra-Application Communications
    Area (ICA) can be used by any program for any purpose. Usually it
    is used to pass data between two or more programs. Not many
    programs use this area. If you wish to use this area, make sure
    checksums and signatures are used to insure the reliability of
    your data, since another program may also decide to use this area.
