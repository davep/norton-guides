 FT_PICKDAY()
 Picklist of days of week
──────────────────────────────────────────────────────────────────────────────

 Syntax

      FT_PICKDAY() -> cDayOfWeek

 Arguments

     None

 Returns

     Character string containing day of week

 Description

     This function is ideal if you need the user to select a day.

 Examples

     mday := FT_PICKDAY()
