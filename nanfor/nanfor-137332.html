 FT_PEGS()
 FT_PEGS GAME (all work and no play...)
──────────────────────────────────────────────────────────────────────────────

 Syntax

      FT_PEGS() -> NIL

 Arguments

     None

 Returns

     NIL

 Description

     This function can be used to alleviate boredom.  The object is to
     remove all pegs except one.  This is done by jumping over adjacent
     pegs.

 Examples

     FT_PEGS()
