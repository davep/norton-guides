FBLD source

        Logic   push source into ST

    FBLD converts a memory operand from BCD (packed decimal) format to
    extended real and pushes it into ST. The sign of the memory operand
    is preserved.


    Note
    ST(7) must be empty; otherwise, the CPU receives an invalid
    operation exception.


    Opcode      Format
    D8 /4       FBLD m80
