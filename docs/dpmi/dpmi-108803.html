        Reset Debug Watchpoint


                 This function  resets the state of a previously defined
                 debug watchpoint.

            To Call

                 AX = 0B03h
                 BX = Debug Watchpoint Handle

            Returns

                 If function was successful:
                 Carry flag is clear

                 If function was not successful:
                 Carry flag is set

            Programmer's Notes

                 None
