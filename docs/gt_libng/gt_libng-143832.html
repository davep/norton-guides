 Sum the ascii values in a string.
──────────────────────────────────────────────────────────────────────────────

 Syntax

        GT_AsciiSum(<cStr>) --> nSum

 Arguments:

      <cStr>  - The string to sum

 Returns:

      nSum    - The sum of all ascii values in <cStr>.

 Description:

      Sum the ascii value of every character in the passed string
      and return the result.
