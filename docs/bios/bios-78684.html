Int 10h, 1114h          Load ROM BIOS 8x16 Alpha, Adjust Height           VGA

    Changes font to ROM BIOS 8x16 font in text mode and adjusts the
    displayed character height.

        Entry   AX = 1114h
                BL = Font block to load (0..7)

        Return  Nothing
