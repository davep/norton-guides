Int 15h, 41h            Wait for External Event                   Convertible

    This service waits for a specified status change after an external
    event. The external event can be an interrupt or the DMA
    controller. While waiting for the specific status, the system
    clocks are stopped in order to conserve battery power.

    This service is available only for the PC Convertible.
