Int 10h, 12h /BL=31h    Select Default Palette Loading              MCGA, VGA

    Enables or disables default palette loading on mode change.

        Entry   AH = 12h
                AL = 0 | 1   Enable | disable default palette loading
                BL = 31h

        Return  AL = 12h   If function supported


    This service selects whether the palette registers, the overscan
    register, and color registers will be loaded with default values
    during subsequent mode sets ("palette lock").
