Int 10h, 1120h          Load User 8x8 Graphics Font            EGA, MCGA, VGA

    Loads high portion of user-defined 8x8 font in graphics mode.

        Entry   AX    = 1120h
                ES:BP = Pointer to character bitmaps
                        (characters 80h..0FFh)

        Return  Nothing


    Note
    Int 1Fh is set to this pointer.
