0040h:0019h           Alt-Keypad Entry

    Length: 1 byte

    This byte contains the progressive input of holding down the Alt
    key and entering numbers from the numeric keypad. As soon as the
    Alt key is released, then the numeric value stored here is stuffed
    into the keyboard buffer and this byte is set to 0.
