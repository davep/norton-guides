Int 10h, 1012h          Update Block of DAC Registers               MCGA, VGA

    Sets the value of one or more consecutively numbered color (DAC)
    registers.

        Entry   AX    = 1012h
                BX    = Number of first color register
                CX    = Number of registers
                ES:DX = Address of table of red-green-blue values
                        (6-bit values)

        Return  Nothing
