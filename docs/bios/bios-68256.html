Int 10h, 0Fh            Get Current Video Mode

    Reports the current video mode, the number of character positions
    per line in that video mode, and the current display page.

        Entry   AH = 0Fh

        Return  AL = Video mode
                AH = Number of character columns per line
                BH = Active display page number
