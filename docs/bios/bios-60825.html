Int 10h, 01h            Set Cursor Size

    Adjusts the size of the cursor by setting its start and end lines.

        Entry   AH = 01h
                CH = Cursor start line (bits 0-4)
                CL = Cursor end line   (bits 0-4)

        Return  Nothing
