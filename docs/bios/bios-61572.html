Int 10h, 03h            Read Cursor Position and Size

    Reports the cursor position (row and column) and size for a
    specified display page.

        Entry   AH = 03h
                BH = Display page number

        Return  CH = Cursor start line
                CL = Cursor end line
                DH = Row
                DL = Column
