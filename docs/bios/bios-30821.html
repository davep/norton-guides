0040h:0062h           Current Display Page

    Length: 1 byte

    Contains the current display page number.

    ──────────────────────────────────────────────────────────────────

    Note
    Use Int 10h, Service 0Fh, to obtain this information.

    Use Int 10h, Service 05h, to set this information.
