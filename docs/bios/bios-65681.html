Int 10h, 0Bh            Set Color Palette/Border Color

    Selects color of background or border.

        Entry   AH = 0Bh
                BH = 00h
                BL = 00..1Fh
                     | Background color (graphics)
                     | Border color only (text modes)

        Return  Nothing

    ──────────────────────────────────────────────────────────────────

    Selects color palette for CGA graphics modes 04h or 05h.

        Entry   AH = 0Bh
                BH = 01h
                BL = Palette
                     | 00h   Green, red, brown
                     | 01h   Cyan, magenta, white

        Return  Nothing
