Int 10h, 12h /BL=34h    Select Cursor Emulation                           VGA

    Selects whether BIOS should automatically remap cursor start/end
    according to the current character height in text modes.

        Entry   AH = 12h
                AL = 0 | 1   Enable | disable cursor emulation
                BL = 34h

        Return  AL = 12h   If function supported
