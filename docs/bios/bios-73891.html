Int 10h, 101Bh          DAC Grey-Scale Summing                      MCGA, VGA

    Converts RGB values in the color registers to grey shades with
    same intensity, on the next video mode set.

        Entry   AX = 101Bh
                BX = Number of first DAC
                CX = Number of registers to convert

        Return  Nothing
