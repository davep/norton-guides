0040h:0060h           Cursor End Scan Line and Start Scan Line

    Length: 2 bytes

    Contains the starting and ending scan lines to determine the size
    of the blinking hardware cursor. The ending scan line is stored
    first, followed by the starting scan line.

    ──────────────────────────────────────────────────────────────────

    See Int 10h, Service 03h, for obtaining the cursor start and end
    scan lines.

    See Int 10h, Service 01h, for setting the cursor start and end
    scan lines.
