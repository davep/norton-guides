Int 10h, 101Ah          Get DAC Color Page                                VGA

    Sets the value of one or more video DAC registers.

        Entry   AX = 101Ah

        Return  BL = Paging mode
                     | 00h    4 pages of 64 registers
                     | 01h   16 pages of 16 registers
                BH = Page number
