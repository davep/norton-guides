0050h:0000h           Print Screen Status Byte

    Length: 1 byte

    Contains the status of the last or current Print Screen operation.

    ──────────────────────────────────────────────────────────────────

    Examination of this byte will reveal the status of the Print
    Screen operation (Int 05h). The byte can have the following three
    values:

        Value   Meaning
        00h     Print Screen complete
        01h     Print Screen currently in progress
        FFh     Error occurred during Print Screen
