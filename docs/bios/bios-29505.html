0040h:0050h           Cursor Position (Column, Row) for Display Pages

    Length: 16 bytes

    The cursor positions for each of the eight display pages. The
    cursor position for each page is in the order of column followed
    by row.

        0040h:0050h - Cursor Position Page 0 (column, row)
        0040h:0052h - Cursor Position Page 1 (column, row)
        0040h:0054h - Cursor Position Page 2 (column, row)
        0040h:0056h - Cursor Position Page 3 (column, row)
        0040h:0058h - Cursor Position Page 4 (column, row)
        0040h:005Ah - Cursor Position Page 5 (column, row)
        0040h:005Ch - Cursor Position Page 6 (column, row)
        0040h:005Eh - Cursor Position Page 7 (column, row)


    Use Int 10h, Service 03h, to obtain this information.
