 FT_E2D()
 Convert scientific notation string to a decimal
──────────────────────────────────────────────────────────────────────────────

 Syntax

            FT_E2D( <cNumE> )  -> <nDec>

 Arguments

                <cNumE>   Scientific notation string to convert

 Returns

                <nDec>    Decimal number

 Description

                Given a string in the format  x.yEz, the decimal
                equivalent is returned.

 Examples

                ? FT_E2D( "1.23E1" )
                  -> 12.3

                ? FT_E2D( "-1.235E1" )
                  -> -12.35

                ? ft_d2e( "5.43E-6" )
                  -> 0.0000543

 Source: E2D.PRG
