 FT_DFCLOSE()
 Close file displayed by FT_DISPFILE()
──────────────────────────────────────────────────────────────────────────────

 Syntax

      FT_DFCLOSE() -> NIL

 Arguments

     None

 Returns

     NIL

 Description

     Closes the file opened by FT_DFSETUP()

 Examples

     @ 4,9 TO 11,71

     FT_DFSETUP("test.txt", 5, 10, 10, 70, 1, 7, 15,;
                 "AaBb" + Chr(143), .T., 5, 132, 4096)

     cKey = FT_DISPFILE()

     FT_DFCLOSE()

     @ 20,0 SAY "Key that terminated FT_DISPFILE() was: " + '[' + cKey + ']'
