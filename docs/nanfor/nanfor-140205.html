 FT_ALT()
 Determine status of the Alt key
──────────────────────────────────────────────────────────────────────────────

 Syntax

      FT_ALT() -> lValue

 Arguments

     None

 Returns

     .T. if Alt key is pressed, .F. if otherwise.

 Description

     This function is useful for times you need to know whether or not the
     Alt key is pressed, such as during a MemoEdit().

 Examples

     IF FT_ALT()
        @24, 0 say "Alt"
     ELSE
        @24, 0 say "   "
     ENDIF
