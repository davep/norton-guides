 FT_MXLIMIT()
 Set vertical bounds of mouse using virtual screen coord.
──────────────────────────────────────────────────────────────────────────────

 Syntax

     FT_MXLIMIT( <nX1>, <nX2> ) -> NIL

 Arguments

    <nX1> is the top row limit.

    <nX2> is the bottom row limit.

 Returns

    NIL

 Description

    Set maximum vertical bounds of mouse using virtual screen coordinates.

 Examples

    FT_MXLIMIT( 10, 20 )

 Source: MOUSE1.PRG
