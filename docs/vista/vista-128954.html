   D_RECSET

SUMMARY
   d_recset(REC[,dbn])
   int REC;
   int dbn;

ARGUMENTS
   REC         The record type for which the current occurrence is to be
               set.

   dbn         (optional)The number of the database containing the record

DESCRIPTION
   This function sets the current occurrence of the record to the current
   record.This function is useful if you whish to perform sequential record
   scan(via d_recnext or d_recprev) from the current record regardless of
   how the current record was established.

CURRENCY CHANGES
   None

RETURN CODES
    -3    S_INVREC     Current record is not in same file that
                       contains REC.

EXAMPLE
    /* Scan sequentially from key,starting with key*/
    d_keyfind(INFO_TYPE,&info_key);
    for(d_recset(INFO);db_status == S_OKAY;d_recnext()){
      d_recread(&info);
      print_recorb(&info);
   }
