Enable set/reset       index 01h  GRAPH

Layout  b3-b0  ESR  Enable set / reset

        The bit numbers in this field correspond to the respective
        display planes and to the bit positions in the Set/reset
        register. The display planes corresponding to the bits
        enabled in this field will be modified by the corresponding
        bits in this Set/reset register. This allows up to four
        display planes to be modified with unique data in a single
        operation.
