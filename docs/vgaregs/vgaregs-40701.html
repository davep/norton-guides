DAC state              read 03C7h  write n/a

Layout  b1-b0  STA  DAC state value

- DAC state value (bits 1-0)
        These 2 bits reflect whether a read or write operation is in
        effect.

        0 : A read operation is in effect. The PEL address read
            register was accessed last.
        3 : A write operation is in effect. The PEL address write
            register was accessed last.
