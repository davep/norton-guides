PEL data               read 03C9h  write 03C9h

Layout  b7-b0  DATA Data value

        This register is an 18-bit wide register used to interface the
        host and the 256 color registers. This data register is
        similar to the other data registers except that it is in
        effect 18 bits wide.

        For a description, refer to 'PEL address write register'.
