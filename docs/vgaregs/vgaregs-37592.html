Overscan color         index 11h  ATTRIB

Layout  b5-b0       Screen border color

        In text modes, the screen border color register selects the
        color of the border that surrounds the text display area on
        the screen (overscan).

        Note
        Not all EGA/VGA implementations operate satisfactorily when a
        color other than black is selected for the border region.
