Cursor end             index 0Bh  CRTC

Layout  b6-b5  CSK  Cursor skew
        b4-b0  CE   Cursor end

- Cursor skew (bits 6-5)
        The number of characters to delay the cursor data in order to
        achieve proper synchronization. The binary amount in this
        field is equal to the number of characters to skew.

- Cursor end (bits 4-0)
        The bottom scan line to display per character row for the
        cursor.
