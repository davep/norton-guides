Read map select        index 04h  GRAPH

Layout  b1-b0  RMS  Read map select

- Read map select (bits 1-0)
        Selects which plane to read in Read mode #0.

        This register has no effect when the VGA is in Read mode #1
        corresponding to the color compare mode.
