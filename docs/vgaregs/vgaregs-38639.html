Horiz pixel panning    index 13h  ATTRIB

Layout  b3-b0  HPP  Horizontal pixel panning

- Horizontal pixel panning (bits 3-0)
        The number of pixels to shift the video data horizontally to
        the left. This affects the starting far left pixel and the
        ending far right pixel on the screen. It is used in
        conjunction with the starting address for smooth panning
        operations.
