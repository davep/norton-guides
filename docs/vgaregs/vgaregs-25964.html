Set/reset reg.         index 00h  GRAPH

Layout  b3-b0  S/R  Set / reset

        The bit positions of this field correspond to the respective
        display planes and to the bit positions in the Enable
        set/reset register. The display planes that correspond to the
        bits enabled in the Enable set/reset register will be modified
        by the corresponding bits in this Set/reset register. This
        allows up to four display planes to be modified with unique
        data in a single operation.
