Int 2Fh / B000h         Get GRAFTABL.COM Installed State             DOS 3.3+

    Determines whether the resident portion of the GRAFTABL command
    has been loaded.

        Entry   AX = 0B000h

        Return  AL = 0FFh | 00h   GRAFTABL loaded | not loaded
