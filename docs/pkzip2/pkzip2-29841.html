  -!   Encrypt authenticity verifiable data into the .ZIP file

  Using this command option will cause PKZIP to embed Authenticity
  Verification (AV) information within the .ZIP file it is creating.

  This option may only be used if you have received and installed the AV
  information for your fully registered copy of PKZIP from PKWARE Inc.

  Note that AV codes for PKZIP 2.0 and above are different than those used
  for version 1.10.

  Please see the section on Authenticity Verification for complete
  information.
