  Using PKZIP and DPMI

  If you experience difficulties when using PKZIP in a DOS window under
  Windows, use the -) option to disable DPMI support.  If this eliminates
  the problem, you may default this option for PKZIP by using the
  configuration file (see manual for details).  With PKUNZIP, this option
  may be set as an environment variable (see below for details).
