  ERROR MESSAGES

  The following error messages may appear when using PKZIP.  When the word
  "WARNING" appears in the message, program execution will continue.
  Otherwise the program will abort to DOS.

  Each warning message is prefaced with an error code.  This error code
  consists of the letter 'E' or 'W' and a number.  The letter corresponds to
  whether the error indicates a "Warning" or "Error" condition.  The number
  corresponds to the DOS error level that would be returned.

  Errors are listed in order of return code or warning level.
