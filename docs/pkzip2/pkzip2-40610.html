  -n   Extract only newer files

  This option is the compliment of the -u option in PKZIP.  When this option
  is used files will only be extracted if:

  -> They do not exist already in the target directory

  OR

  -> They are newer than the file already in the target directory

  This means that files which already exist and are more recent than those
  in the .ZIP file will not be overwritten.
