  -i   Add files that have the archive attribute - then turn attribute off
  -i-  Add files that have the archive attribute - do not turn attribute off

  This option is very handy when backing up files.  If this option is
  specified, only files with the archive attribute will be compressed, and
  the archive attribute will be set to off when the ZIP operation is
  complete for these files.

  You may alternatively ZIP all files with the archive attribute set, but
  not modify the archive attribute.

  A detailed explanation of the use of this option is given in the PKZIP
  Backup Guide.
