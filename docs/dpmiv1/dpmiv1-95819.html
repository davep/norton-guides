Get Page Size

Returns the size of a single memory page in bytes.

Call With:

   AX     = 0604H

Returns:

   if function successful
   Carry flag= clear
   BX:CX  = page size in bytes

   if function unsuccessful
   Carry flag= set
   AX     = error code
           8001H   unsupported function (16-bit host)
