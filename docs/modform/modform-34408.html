 Subfunction 27 : Play loaded module

 AH = 82
 AL = 27
 This function is used to play the module loaded with subfunction 24, and
 to set various options applicable to the module. BX determines the operation
 that is to be performed. AX will probably be set to 5722 on return regardless
 of the validity of the operation.

    ┌────┬───────────────────────────────────────────────────┐
    │ BX │ Action                                            │
    ├────┼───────────────────────────────────────────────────┤
    │ 1  │ Set the default speed back to 6.                  │
    │ 2  │ Set the output device number to the value in CL.  │
    │ 3  │ Play the module.                                  │
    └────┴───────────────────────────────────────────────────┛
