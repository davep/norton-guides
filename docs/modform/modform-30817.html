 Subfunction 22 : Reset the interrupt vector

 AH = 82
 AL = 22
 This function will set interrupt 2F back to its original value. Note that
 MODRES is not removed from memory - just disabled. AX will hold the segment
 of the PSP on return, and can thus be used to remove MODRES from memory.
