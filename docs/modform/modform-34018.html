 Subfunction 24 : Load a module

 AH = 82
 AL = 24
 DX:CX points to a 20 character null terminated file name.
 This function will attempt to load the module, specified by the filename,
 from disk. If the function is successful, AX will hold 5722h on return.
 This function does not play the module - function 27 should be used for
 that.
