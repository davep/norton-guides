 Soundblaster Instrument File (SBI)

 An SBI file contains FM parameters for one instrument. The format is

 ──────────┬───────────────────────────────────────────────────────
   Offset  │  Usage
 ──────────┼───────────────────────────────────────────────────────
    00-03  │  File ID - 'SBI' + 1Ah
    04-23  │  Instrument Name
    24-33  │  FM information as defined in "Instrument Block"
 ──────────┴───────────────────────────────────────────────────────
