    Transfer Lines command

    Purpose:  Transfers (merges) the contents of a specified file into the
              file currently being edited.

    Format:   [line]T[d:]filename

    Remarks:  The filename contents are inserted ahead of the line in
              the file being edited. If line is omitted, the current line
              is used.

                 Note:
                 The file being merged is read from the current directory
                 of the specified or default drive. If a path was specified
                 when you issued the EDLIN command, that path becomes the
                 current directory for that drive for the duration of the
                 EDLIN session, and any Transfer Lines commands for that
                 drive must be satisfied from the same directory.
