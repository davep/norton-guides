    Quit Edit command

    Purpose:  Quits the editing session without saving any changes you may
              have entered.

    Format:   Q

    Remarks:  EDLIN prompts you to make sure you really don't want to save
              the changes.

              Enter Y if you want to quit the editing session. No editing
              changes are saved and no .BAK file is created. Refer to the
              End Edit command for information about the .BAK file.

              Enter N, or any other character, if you want to continue
              the editing session.

    Example:  When you enter the Q command, the following is displayed:

              Q
              Abort edit (Y/N)? _
