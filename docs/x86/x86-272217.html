FLD source

        Logic   push source into ST

    FLD pushes source onto the FPU stack and is the basic instruction
    for moving data into the FPU.
    If the source is a register, the register number used is that before
    the stack-top pointer is decremented. If source is in single or
    double FP format, it is automatically converted to extended real
    format.

        Example
        fld   [mem64]   ; load double
        fld   ST(0)     ; duplicate stack top


    Note
    ST(7) must be empty; otherwise, the CPU receives an invalid
    operation exception.


    Opcode      Format
    D9 /0       FLD m32
    DD /0       FLD m64
    DB /5       FLD m80
    D9 C0 + i   FLD ST(i)
