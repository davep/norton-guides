FBSTP destination

        Logic   destination ← ST
                pop ST

    FBSTP converts the value in ST into a packed decimal integer (BCD),
    and stores the result at the specified memory destination. It then
    pops the stack.
    If ST contains a non-integral value, FBSTP rounds it according to
    the RC field of the control word.


    Opcode      Format
    DF /6       FBSTP m80
