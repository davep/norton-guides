FDECSTP

        Logic   TOP ← (TOP - 1) AND 07h

    FDECSTP substracts one from the three-bit TOP field of the status
    word. In effect, FDECSTP rotates the stack; it is not equivalent to
    pushing the stack. If TOP is 0, this instruction makes it 7.


    Opcode      Format
    D9 F6       FDECSTP
