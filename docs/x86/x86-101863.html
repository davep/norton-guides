NOT             NOT operation                        Flags: Not altered

NOT destination

        Logic   destination ← NOT(destination)  ; One's complement

    NOT inverts each bit in its operand, i.e. forms the one's
    complement.

                       NOT truth table
                        a       NOT a
                        0         1
                        1         0


    Opcode      Format
    F6 /2       NOT  r/m8,r8
    F7 /2       NOT  r/m16,r16
    F7 /2       NOT  r/m32,r32
