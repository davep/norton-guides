        INDENT_LOCALS

    Default: no

    Options: YES, NO

    This directive is only operative when INDENT_FUNCTIONS is set to
    YES.

    If INDENT_FUNCTIONS=YES, and INDENT_LOCALS=NO, then any line
    starting with LOCA PRIV MEMV STAT PUBL FIEL is left justified.
