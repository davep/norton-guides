    Input command

    Purpose:  Inputs and displays (in hexadecimal) one byte from the speci-
              fied port.

    Format:   I portaddress

    Example:  I 2F8
              6B

              The single hexadecimal byte read from port 02F8h is displayed
              (6Bh).
