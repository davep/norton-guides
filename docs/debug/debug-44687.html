  Insufficient space on disk

    A Write command was issued to a disk that doesn't have enough free
    space to hold the data being written.

    If you are writing to a diskette, you can insert a diskette that has
    enough free space, then reissue the Write command. Otherwise, you
    should erase files from the disk and run DEBUG again.
