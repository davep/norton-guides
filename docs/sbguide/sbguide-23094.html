Block type - 7 REPEAT END BLOCK.

This BLKTYPE specifies the end of a repeat loop. When this block
type is reached the output process repeats from the sub-block
after the BLKTYPE again.

Application programs may break-out of this repeat loop and continue
from the sub-block after this block by invoking a function 12 call to
the CT-VOICE.DRV.

The BLKLEN is fixed at 0.
