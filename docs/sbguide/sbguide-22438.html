Block Type - 6 REPEAT BLOCK.

This BLKTYPE specifies the beginning of a repeat loop. The data
sub-blocks in between this block and the BLKTYPE - 7 will be
repeated based on the value of the count specified.

The COUNT is a value from 1 to 0FFFFG. It indicates the number
of time the voice is to be repeated. 0FFFFH indicates that the voice
is to be repeated non-stop.

The actual time it will play is COUNT + 1,
eg. If COUNT = 3, the data sub-blocks are output 4 times.

The BLKLEN is fixed at 2.
