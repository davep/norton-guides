Function 11     : Continue output voice.

Entry          : BX = 11.
Exit           : AX = 0, Succesful.
                 AX = 1, No voice to continue.

The paused voice output process is continued. The status word
will be updated with the marker in the CVFF format or a zero at the
end of voice output.
