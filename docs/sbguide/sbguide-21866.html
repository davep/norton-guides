Block Type - 5 MESSAGE BLOCK

This block type specifies an ASCII string in the voice data file. The
ASCII string is terminated with a NULL character (ASCII 0).

Most application programs may choose to ignore this BLKTYPE.
Is is defined here, for an application program which requires
ASCII information on the voice. For example, name, type or
contents of the voice.
