Int 14 Fn E6  - Telapi - "telunload" - Uninstall                           [N]

   AH = E6h

Return: AX = status
       0000h successful
       FFFFh unable to uninstall

Notes: TelAPI also supports the NASI/NACS and NCSI APIs on INT 6B
   this function invokes AH=E5h internally

See Also: AH=E5h,AX=FF00h,INT 6B/AH=00h,INT 6B/AH=10h
