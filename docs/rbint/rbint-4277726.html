Int 2F Fn B80F  - Dos Lan Requester - Get Start Parameters                 [N]

   AX = B80Fh
   CX = size of return data buffer
   ES:DI -> return data buffer (see #2668)

Return: AX = status
        00h     network started
        nonzero network not started
   CX = number of bytes returned in buffer
   ES:DI buffer filled

See Also: AX=B809h"PC LAN Program",AX=B80Eh
