Int 2F Fn B710  U - Dos 3.3+ Append - Get Version Info                     [f]

   AX = B710h

Return: AX = current APPEND state (see #2664)
   BX = ??? (0000h in MS-DOS 3.30 and 5.00)
   CX = ??? (0000h in MS-DOS 3.30 and 5.00)
   DL = major version
   DH = minor version

See Also: AX=B700h,AX=B702h
