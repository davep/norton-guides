Int 7F Fn 4E  - Alloy Mw386 V2+ - Set Error Mode                           [N]

   AH = 4Eh
   AL = error mode flags
       bit 0: display critical disk errors
       bit 1: display sharing errors
   DX = 4E58h ("NX")

Return: AL = status
       00h successful

See Also: AH=4Fh
