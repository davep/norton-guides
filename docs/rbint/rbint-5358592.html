Int 62 Fn 00DE  u - Fgdriver V4.02 - "fg_waitkey" - Flush Keyboard Buffer  [V]

   AX = 00DEh

Return: after next key pressed

Note:  the FGDRIVER functions are rearranged with each major release, but
     their parameters do not change (see #3145 at AX=0000h)

See Also: AX=000Ah,AX=0046h,AX=0063h,AX=0069h,INT 16/AH=00h
