Int 17 Fn 5001  - Ax (japanese At) Printer - Get Printer Country Code      [J]

   AX = 5001h

Return: AL = status
       00h successful
       BX = country code
       02h error

See Also: AX=5000h,AH=51h,INT 10/AX=5001h,INT 16/AX=5001h
