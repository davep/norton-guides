Int 10 Fn BF07  - Video - Compaq Slt/286 - Write Gray Scale Table          [b]

   AX = BF07h
   CH = value to write to gray scale table
   CL = address to be written to gray scale table

Return: nothing

See Also: AX=BF06h,AX=BF08h
