Int 14 Fn 0F  - Mbbios Paccom Support - Set Slot Time                      [S]

   AH = 0Fh
   AL = new setting in clock ticks

Return: nothing
Desc:  specify the time from end of DCD to RTS

See Also: AH=0Dh"MBBIOS",AH=0Eh"MBBIOS",AH=10h"MBBIOS"
