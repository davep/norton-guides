Int 2F Fn CB0A  - Communicating Applications Specification - Get Event Da  [F]

   AX = CB0Ah
   BX = event handle
   DL = queue
       00h task queue
       01h receive queue
       02h log queue

Return: AX = 0000h successful
       CX = year
       DH = month
       DL = day
   AX < 0     error code (see #2720)

See Also: AX=CB0Bh,AX=CB0Ch
