Int D4 Fn 27  - Pc-mos/386 V5.01 - Get Task Control Block Segment/selecto  [O]

   AH = 27h
   BX = task ID (FFFFh if current task) (see AH=19h)

Return: CF clear if successful
       DX = segment/selector for the Task Control Block (see #3677)
   CF set on error
       AX = error code (see #1366 at INT 21/AH=59h/BX=0000h)

Note:  this function supersedes AH=04h

See Also: AH=26h,AH=28h,AH=29h,AH=2Ah
