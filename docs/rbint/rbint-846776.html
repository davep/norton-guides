Int 13 Fn 38  U - Quickcache Ii V4.20 - Set Buffered Writes For Specific   [c]

   AH = 38h
   AL = new state (00h disabled, else enabled)
   DL = drive number (00h = A:)

Return: AX = status (0000h successful)

See Also: AH=2Ch,AH=37h
