Int 13 Fn 1F  - Syquest - Door Latch/door Button Detect                    [d]

   AH = 1Fh
   AL = subfunction
       00h allow media removal
       01h prevent media removal (lock door)
   DL = drive ID (bit 7 set for hard disks)

Return: CF clear if successful
       AH = 00h
   CF set on error
       AH = error code
       00h successful
       01h invalid function request
       80h timeout
       DDh media change requested

See Also: AH=12h"SyQuest",AH=13h"SyQuest",AH=59h"SyQuest"
