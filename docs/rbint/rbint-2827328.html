Int 21 Fn C702  - Novell Netware - Transaction Tracking System - Installa  [N]

   AX = C702h

Return: AL = status
       00h not available
       01h available
       FDh available but disabled
Desc:  determine whether the default file server supports TTS

Note:  this function is supported by NetWare 4.0+ and Advanced NetWare 1.0+

See Also: AX=C700h,AX=C701h,AX=C703h,AX=F222h/SF=00h
