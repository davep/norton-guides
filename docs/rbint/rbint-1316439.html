Int 15 Fn 75  - Hunter 16 - Set Contrast Power Up State                    [b]

   AH = 75h
   AL = new level
       01h off
       02h unchanged
       03h change level
       BL = desired level (00h-7Fh)

Return: AH = status
Desc:  select the LCD contrast level to use when the Hunter is next turned on

See Also: AH=63h,AH=74h
