Int 2F Fn 5482  U - Power.exe - Get/set Apm Polling Frequency              [p]

   AX = 5482h
   BX = new polling frequency or 0000h to get current frequency

Return: AX = 0000h (successful)
   BX = current frequency if BX=0000h on entry

See Also: AH=53h,AX=5400h,AX=5401h,AX=5480h,AX=5481h,AX=548Fh
