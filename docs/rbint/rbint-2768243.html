Int 21 Fn 71A8  - Windows95 - Long Filename - Generate Short Filename      [D]

   AX = 71A8h
   DS:SI -> ASCIZ long filename (no path allowed!)
   ES:DI -> buffer for ASCIZ short filename
   DH = short name's format
       00h 11-char directory entry/FCB filename format
       01h DOS 8.3
   DL = character sets
       bits 7-4: short name's character set (see #1474)
       bits 3-0: long name's character set (see #1474)

Return: CF clear if successful
       ES:DI buffer filled
   CF set on error
       AX = error code
       7100h if function not supported

Note:  this function uses the same algorithm as the filesystem except that
     the returned name never has a numeric tail for disambiguation

See Also: AX=7160h/CL=00h,AX=7160h/CL=02h,AX=71A7h/BL=00h
