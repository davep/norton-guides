Int 15 Fn 7200  - Hunter 16 - Reset Serial Number                          [b]

   AX = 7200h

Return: AH = 00h if successful
Desc:  reset the serial number to the default serial number present when the
     Hunter 16 was shipped

See Also: AH=6Ch,AH=72h
