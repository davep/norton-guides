Int 15 Fn B001  - Stac ??? - Installation Check                            [-]

   AX = B001h
   CX = 5354h ('ST')
   DX = 4143h ('AC')

Return: AX = 4F4Bh ('OK') if installed

Note:  this function is called by Novell DOS 7 NWCACHE v1.01
