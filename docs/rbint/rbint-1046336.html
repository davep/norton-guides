Int 14 Fn E001  - Mx5 Extended Fossil - Get/set Mnp Level                  [S]

   AX = E001h
   BH = function
       00h get MNP level
       01h set MNP level
       BL = new level (00h none, 02h/04h/05h MNP level N)
   DX = port number (0-3)

Return: BL = MNP level

See Also: AX=E002h,AX=E003h,AX=E004h,AX=E006h
