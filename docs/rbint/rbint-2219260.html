Int 21 Fn 37A6  - Xpack V1.65 - Get Temporary Directory Name               [-]

   AX = 37A6h
   BX = 6A6Dh ('jm')

Return: AX = 0000h
   DS:DX -> name of temporary directory
Program: XPACK is a transparent file compressor/decompressor by JauMing Tseng
