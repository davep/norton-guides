Int 11 Fn 0225  - Blank-it Screen Blanker - Installation Check             [V]

   AX = 0225h
   BX = 6900h

Return: BL = 23h
   ES:DI -> ASCIZ "BLNKIT"
Program: Blank-It is a resident screen blanker by Rhode Island Soft Systems,
     Inc.
