Int 2F Fn AF0D  - Windos V2.11 - Get Current Parameter                     [O]

   AX = AF0Dh
   BX = offset of desired parameter in global data area

Return: AX = current value of parameter (AH undefined if byte value)

See Also: AX=AF00h"WinDOS",AX=AF0Eh
