Int 2F Fn C302  - Setwpr.com - Set Write-protection State                  [d]

   AX = C302h
   BL = new state
       00h disk may be written
       01h disk write-protected

Return: AL = FFh if installed
Program: SETWPR is a hard-disk write-protection TSR by Jaroslaw Rafa
