Int 18 Fn 01  - Nec Pc-9800 Series - Keyboard - Check For Keystroke        [J]

   AH = 01h

Return: BH = status
       00h no keystrokes available
       01h keystroke available
       AX = keystroke

See Also: AH=00h,AH=02h,INT 16/AH=01h
