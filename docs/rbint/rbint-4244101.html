Int 2F Fn AD02  U - Dos 3.3+ Display.sys Internal - Get Active Code Page   [V]

   AX = AD02h

Return: CF set if code page never set
       AX = 0001h
       BX = FFFFh (assume first hardware code page)
   CF clear if successful
       BX = current code page (see #1446 at INT 21/AX=6602h)
