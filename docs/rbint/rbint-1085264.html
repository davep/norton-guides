Int 15 Fn 03  - Cassette - Write Data (pc And Pcjr Only)                   [B]

   AH = 03h
   CX = number of bytes to write
   ES:BX -> data buffer

Return: CF clear if successful
       ES:BX -> byte following last byte written
   CF set on error
   AH = status (see #0335)
   CX = 0000h

See Also: AH=00h"CASSETTE",AH=02h"CASSETTE"
