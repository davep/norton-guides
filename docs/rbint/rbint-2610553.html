Int 21 Fn 5E01  - 10net V5.0 - Get Local 10net Configuration Table         [N]

   AX = 5E01h
   CX = length of buffer
   DS:DX -> buffer for 10Net configuration table (see #1377)

Return: DS:DX buffer filled

See Also: AX=5E00h"10NET",INT 6F/AH=02h,INT 6F/AH=03h
