Int 6F Fn 10  - Hp Vectra Ex-bios - "f_put_sprite" - Display Graphics Cur  [b]

   AH = 10h
   BP = 005Ah (driver ID for V_STRACK)
   BX = X coordinate
   CX = Y coordinate

Return: AH = status (see #3415)
   BP,DS destroyed

See Also: AH=00h"HP Vectra",AH=04h"F_TRACK_INIT",AH=08h"F_TRACK_OFF"
