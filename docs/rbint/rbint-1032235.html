Int 14 Fn A2  - 3com Bapi Serial I/o - Write Character                     [S]

   AH = A2h
   AL = character
   DH = session ID (00h for external session managment)

Return: AH = return code (00h,01h,03h,07h,0Ah,0Bh) (see #0320)

See Also: AH=A0h"BAPI",AH=A3h"BAPI",AH=A4h"BAPI"
