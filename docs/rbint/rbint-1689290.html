Int 17 Fn CD09  - Inset - Alter Text Line Spacing                          [P]

   AX = CD09h
   CX = line spacing in 1/720th inch

Return: CX = 07C2h

Note:  not yet implemented, line spacing is currently fixed at 1/6 inch

See Also: AX=CD06h
