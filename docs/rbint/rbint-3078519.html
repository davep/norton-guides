Int 21 Fn F004  - Novell Netware - Workstation - Set Primary Connection I  [N]

   AX = F004h
   DL = connection ID of primary file server (1-8) or 00h for none

Note:  this function is supported by Advanced NetWare 2.0+

See Also: AH=D6h,AX=EF03h,AX=F000h,AX=F002h,AX=F005h
