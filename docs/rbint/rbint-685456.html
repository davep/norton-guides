Int 10 Fn 7F01  - Sollex Supervga - Adapter Control - Get Adapter          [V]

   AX = 7F01h
   BL = 01h

Return: AL <> 7Fh if not supported
   AL = 7Fh if supported
       AH = status
       00h successful
       01h failed
       BX = adapter type (see #0137)
       DX = display type (see #0138)

See Also: AX=7F00h"SOLLEX",AX=7F01h/BL=00h,AX=7F01h/BL=02h
