Int 2F Fn 7F01  - Jim Harper's Cd-rom Redirector Scsi Driver - Do Command  [d]

   AX = 7F01h
   DS:DX -> command record (see #2613)

Return: AL = status
       00h successful
       else error code

See Also: AX=7F00h"Harper",AX=7F02h"Harper",INT 11/AH=FFh"SDLP"

See Also: INT 21/AX=4402h"ASPI",INT 4F/AX=8100h

Format of CD-ROM redirector command record:
Offset Size    Description (Table 2613)
 00h   BYTE    ID
 01h 10 BYTEs  CDB (Command Descriptor Block) for operation (see #2907,#2908)
 0Bh   WORD    segment of buffer
 0Dh   WORD    offset of buffer
 0Fh   BYTE    status
 10h   BYTE    sense
 12h   WORD    count
