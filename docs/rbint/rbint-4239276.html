Int 2F Fn AC18  - Qrip/tsr - Send User Key To Qrip                         [V]

   AX = AC18h
   DI = 1092h
   BX = key (00xxh is normal ASCII, xx00h is an extended key)

Return: nothing

Note:  if DI <> 1092h on entry, QRIP chains the call

See Also: AX=AC06h,AX=ACF0h
