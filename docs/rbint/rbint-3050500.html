Int 21 Fn E3  - Novell Netware - File Server - Get File Server Lan I/o St  [N]

   AH = E3h subfn E7h
   DS:SI -> request buffer (see #1767)
   ES:DI -> reply buffer (see #1768)

Return: AL = status
       00h successful

Note:  this function is supported by Advanced NetWare 2.1+

See Also: AH=E3h/SF=0Eh,AH=E3h/SF=11h,AH=E3h/SF=D3h,AH=E3h/SF=E8h

See Also: AH=E7h"Novell",AX=F217h/SF=E7h

Format of NetWare "Get File Server LAN I/O Statistics" request buffer:
Offset Size    Description (Table 1767)
 00h   WORD    0001h (length of following data)
 02h   BYTE    E7h (subfunction "Get File Server LAN I/O Statistics")

See Also: #1768,#1933 at AX=F217h/SF=E7h
