Int E0  - Dr Multiuser Dos - "a_writeblk" - Write String To Aux Device     [O]

   CL = ADh
   DS:DX -> character control block (see #3700,#3787)

Return: AX = number of characters written

Notes: does not return until at least one character has been written
   also supported by REAL/32

See Also: INT E0/CL=04h,INT E0/CL=A5h,INT E0/CL=ACh
