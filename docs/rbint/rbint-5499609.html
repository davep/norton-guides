Int 67 Fn 40  - Lim Ems - Get Manager Status                               [m]

   AH = 40h

Return: AH = status (00h,80h,81h,84h) (see #3319)

Note:  this call can be used only after establishing that the EMS driver is in
     fact present

See Also: AH=3Fh,AX=FFA5h,@xxxxh:xxxxh"PMM"
