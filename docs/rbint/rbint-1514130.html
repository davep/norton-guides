Int 16 Fn 24  - Hunter 16 - Redefine Key Codes                             [b]

   AH = 24h
   AL = Matrix Code (see #0515)
   BL = new Key code

Return: AL = status (00h successful, nonzero failed)

See Also: AH=2Bh,AH=2Ch
