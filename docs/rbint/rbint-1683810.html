Int 17 Fn AA  - Alloy Mw386 V2+ - Get Real-time Printer Status             [N]

   AH = AAh
   AL = mode
       00h NTNX
       DX = NTNX printer number (see #0567)
       01h MW386
       DX = MW386 printer number

Return: AH = instantaneous printer status
       00h printer ready
       01h not ready
       12h off line
       13h out of paper
       14h general device failure
       15h device timeout
       16h bad device number
