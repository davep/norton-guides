Int 18 Fn 690D  - Gravis Ultra Sound Yea_gus.exe - Restart Voice           [s]

   AX = 690Dh
   BX = voice
   CX = sample type (0 = 8-bit, 1 = 16-bit)
   DX = looping type (0 = none, 1 = forward, 2 = back and forth)

Return: CX = balance value

See Also: AX=6907h,AX=6908h
