Int 1A Fn 80  - Pcmcia Socket Services V1.00 - Get Number Of Adapters      [X]

   AH = 80h

Return: CF clear if successful
       CX = 5353h ('SS') if Socket Services installed
       AL = number of adapters present (0-16)
       AH destroyed
   CF set on error
       AH = error code (see #0580)

See Also: AH=83h"PCMCIA"
