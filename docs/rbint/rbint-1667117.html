Int 17 Fn 5000  - Ax (japanese At) Printer - Set Printer Country Code      [J]

   AX = 5000h
   BX = country code
       0001h USA (English), 0051h Japan

Return: AL = status
       00h successful
       01h bad country code
       02h other error

See Also: AX=5001h,AH=51h,INT 10/AX=5000h,INT 16/AX=5000h
