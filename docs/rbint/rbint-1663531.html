Int 17 Fn 2400  - Shamrock Software Net.24 V3.11+ - Enable/disable Api Fu  [N]

   AX = 2400h
   DL = new state
       00h disabled
       01h enabled

Return: DL = 24h if installed
   DH = minor version number
   CX = network address of this machine
   AL = status (see #0564)

See Also: AX=2403h,INT 16/AX=4500h

(Table 0564)
Values for NET.24 function status:
 00h   successful
 01h   timeout
 02h   header error
 03h   data error
 04h   busy
 05h   invalid parameters
