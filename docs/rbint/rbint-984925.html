Int 14 Fn 6F  - Connection Manager - Installation Check                    [N]

   AH = 6Fh
   BX = FFFFh

Return: DX:BX -> Connection Manager Communication Table if installed
   BX = FFFFh if not installed

See Also: AH=0Dh/DX=FFFFh
