Int 21 Fn DF09  - Software Carousel - Boot The System                      [T]

   AX = DF09h

Return: AL = status
       00h Carousel not running
       FFh unsucessful
       AH = error code (01h,03h) (see #1501)

Note:  this function never returns if successful
