Int 32 Fn 6E16  - Noise.sys V0.53+ - Read Urandom Bytes                    [y]

   AX = 6E16h
   CX = number of bytes
   ES:DI -> buffer

Return: CF set on error
       AL = error code (FDh,FEh) (see #2837)
   CF clear if successful
       CX = number of random bytes read

See Also: AX=6E00h,AX=6E12h,AX=6E17h
