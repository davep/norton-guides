Int 16 Fn AF51  - K3plus V6.00+ (api V2.00+) - Set Options                 [K]

   AX = AF51h
   BX = 4B33h ('K3')
   CX = switches

Return: AL = status
       00h done
       CX = previous switch settings
       01h switch not supported
       FFh other error

See Also: AX=AF4Dh,AX=AF50h,AX=AF80h
