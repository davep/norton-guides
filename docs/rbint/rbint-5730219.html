Int 6F Fn 99  - 10net V5.0 - Audit                                         [N]

   AH = 99h
   DS:SI -> server's node ID
   ES:DI -> data to be appended to audit trail file (max 106 bytes)

Return: CF clear if successful
   CF set on error
       AX = status (see #3411)

See Also: AH=19h,AH=9Ch
