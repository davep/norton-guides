Int 6F Fn 06  - Hp Vectra Ex-bios - "f_sample" - Poll Current Absolute Po  [b]

   AH = 06h
   BP = driver ID (see #3412) for V_LTOUCH, V_LPOINTER, V_LTABLET, etc.

Return: AH = status (see #3415)
   BX = current logical X position
   CX = current logical Y position
   DL = button transitions (bit N = button N state has changed)
   DH = current button states (bit N = current button N state)
   ES:0000h -> logical device's Describe Record (see #3420)
   BP,DS destroyed

See Also: AH=00h"HP Vectra",AX=040Ah"SF_EVENT_ON",AX=040Eh"SF_CLIPPING_ON"
