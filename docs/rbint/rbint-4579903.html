Int 2F Fn FF16  - Topware Network Os V5.10+ - Set Control Number Of "show  [N]

   AX = FF16h
   BL = which to set (00h TopShow, FFh TopTerm)
   CX = destination screen
       0000h all stations
       0000h-00FFh (TopTerm only) send to group CL
       8001h-80FEh send to station CL

See Also: AX=FF18h
