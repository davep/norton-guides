Int 2F Fn CD00  R - Interwave Game Api - Installation Check                [s]

   AX = CD00h
   DX = 4957h ('IW')
   SI = 5645h ('VE')

Return: AL = FFh if installed
       DX = 4554h ('ET')
       SI = 454Bh ('EK')
Range: AH=CDh-ECh, selected by scanning for an available multiplex number

See Also: AX=CD00h/BX=464Fh,AX=CD01h"InterWave",AX=CD21h"InterWave"
