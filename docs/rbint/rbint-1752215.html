Int 1A Fn 86  - Pcmcia Socket Services V1.00 - Set Adapter                 [X]

   AH = 86h
   AL = adapter number
   DH = new adapter attributes (see #0596)

Return: CF clear if successful
       AH destroyed
   CF set on error
       AH = error code (01h) (see #0580)

See Also: AH=84h"PCMCIA",AH=85h"PCMCIA"
