Int 7F Fn 0A  - G8bpq V4.00+ - Host Mode - Transmit Raw (kiss) Frame       [N]

   AH = 0Ah
   AL = radio port
   ES:SI -> buffer containing data to be sent
   CX = number of bytes to send

See Also: AH=02h"G8BPQ",AH=08h"G8BPQ",AH=0Bh"G8BPQ"
