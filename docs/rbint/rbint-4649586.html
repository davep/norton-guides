Int 31 Fn FF05  P - Causeway - "codesel" - Convert Selector To Executable  [E]

   AX = FF05h
   BX = selector
   CL = default operation size (00h = 16-bit, 01h = 32-bit)

Return: CF clear if successful
   CF set on error

See Also: AX=FF03h
