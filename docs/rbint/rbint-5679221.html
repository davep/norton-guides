Int 6F Fn 040E  - Hp Vectra Ex-bios - "f_io_ctrl Sf_clipping_on" - Enable  [b]

   AX = 040Eh
   BP = driver ID (see #3412) for V_LPOINTER, V_LTOUCH, V_LTABLET

Return: AH = status (see #3415)
   BP,DS destroyed
Desc:  causes the physical device's motion to be clipped to prevent overflow
     or underflow of the logical position

See Also: AH=00h"HP Vectra",AX=040Ah"SF_EVENT_ON",AX=0410h"SF_CLIPPING_OFF"
