Int 67 Fn 1F  U - Qualitas 386max V7.00 - Memlimit - Api                   [m]

   AH = 1Fh
   DS:SI -> request packet (see #3194)

Return: AH = status (00h successful, 84h invalid function code, etc.)

See Also: AH=1Eh

Format of 386MAX MEMLIMIT request packet:
Offset Size    Description (Table 3194)
 00h   WORD    function code (00h-0Fh)
 02h   WORD    return code (see #3195)
 04h  4 BYTEs  ???
 08h   WORD    ???
   ???

(Table 3195)
Values for 386MAX MEMLIMIT return code:
 00h   unknown request
 01h   invalid parameter for VCPI limit
 02h   VCPI limit set
 03h   invalid parameter for EMS limit
 04h   EMS limit set
 05h   DPMI disabled
 06h   XMS disabled
 07h   XMS limit set
 08h   unable to uninstall
 09h   unloaded
