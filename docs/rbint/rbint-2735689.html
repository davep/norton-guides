Int 21 Fn 6A  U - Dos 4.0+ - Commit File                                   [D]

   AH = 6Ah
   BX = file handle

Return: CF clear if successful
       AH = 68h
   CF set on error
       AX = error code (06h) (see #1366 at AH=59h/BX=0000h)

Note:  identical to AH=68h in DOS 5.0-6.0; not known whether this is the case
     in DOS 4.x

See Also: AH=68h
