Int 21 Fn DF  - Novell Netware - Print Services - End Specific Lpt Captur  [N]

   AH = DFh
   DL = 05h
   DH = LPT port (00h-02h)

Return: AL = status
       00h successful
Desc:  stop redirecting the specified LPT, close the capture file, and release
     the job in the print queue for printing

Notes: this function is supported by Advanced NetWare 2.1+
   after this call, the specified LPT defaults to local printing

See Also: AH=DFh/DL=01h,AH=DFh/DL=04h,AH=DFh/DL=06h,AH=DFh/DL=07h
