Int 10 Fn CD05  - Video - Ultravision - Set Cursor Type (ega,vga)          [V]

   AX = CD05h
   CL = type
       00h line cursor
       FFh box cursor

Return: nothing

Note:  sets default cursor type for text-based programs

See Also: AH=01h,AX=CD06h
