Int 2F Fn CB0B  - Communicating Applications Specification - Set Task Dat  [F]

   AX = CB0Bh
   BX = event handle (task event only)
   CX = year
   DH = month
   DL = day

Return: AX = 0000h successful
   AX < 0     error code (see #2720)

Note:  setting a task's date and time to before the current date and time
     causes it to execute immediately

See Also: AX=CB01h,AX=CB0Ah,AX=CB0Dh
