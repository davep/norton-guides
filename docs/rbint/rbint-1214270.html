Int 15 Fn 18  - Vmix - "sys_setcolors" - Set Console Colors                [T]

   AH = 18h
   STACK:  WORD    new background/foreground colors
           bits 3-0: foreground
           bits 7-4: background
           bits 15-8: unused

Return: AX = color

See Also: AH=14h"VMiX",AH=17h"VMiX",AH=19h"VMiX"
