Int 50 Fn 0004  - Vanderaart Text Windows - Get Key                        [V]

   AX = 0004h
   CH = type
       00h any key
       01h 'J' or 'N' (Dutch for yes/no)

Return: AX = key

See Also: INT 16/AH=00h
