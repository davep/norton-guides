Int 2F Fn 1AAA  - Ansiplus V3.01+ - Get/set Ansiplus Screen Saver Blankin  [V]

   AX = 1AAAh
   BX = function
       FFFFh to get current blanking time
       other to set time
       CX = blanking time in clock ticks (0000h-7FFFh)

Return: BX = current blanking time
   CX = blanking time when last set

See Also: AX=1A00h/BX=414Eh,AX=1AABh
