Int 10 Fn 1101  - Video - Text-mode Chargen - Load Rom Monochrome Pattern  [V]

   AX = 1101h
   BL = block to load

Return: nothing

Notes: (see also AX=1100h)
   This function will cause a mode set, completely resetting
     the video environment, but without clearing the video buffer
   the "monochrome" patters are 8x14 pixels in size

See Also: AX=1100h,AX=1102h,AX=1103h,AX=1104h,AX=1111h,AH=1Bh,AX=CD10h
