Int 15 Fn 53B0  - Intel System Management Bus - Disconnect                 [X]

   AX = 53B0h
   BH = 05h
   CX = 6941h ('iA')

Return: CF clear if successful
       AH = 00h (SMBus OK)
   CF set if error
       AH = error code (03h,04h,05h,0Ah,86h) (see #0410)

Note:  this function is supported in connected mode (far CALL entry point)
     only

See Also: AX=53B0h/BH=01h,AX=53B0h/BH=02h,AX=53B0h/BH=03h,AX=53B0h/BH=04h
