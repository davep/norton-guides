Int 10 Fn 1120  - Video - Graph-mode Chargen - Set User 8x8 Graphics Char  [V]

   AX = 1120h
   ES:BP -> user table for INT 1F

Return: nothing

Note:  this function is meant to be called immediately after a mode set;
     results are unpredictable at other times

See Also: AX=1121h,AX=1122h,AX=1123h,AX=1124h,AX=1129h,INT 1F"SYSTEM DATA"
