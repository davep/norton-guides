Int 21 Fn FFBB  - Topware Network Os V5.10+ - Get Printer Server Station   [N]

   AX = FFBBh
   CH = printer number (00h LPT1, 01h LPT2, 02h LPT3)

Return: AL = current mapping printer server station number
       00h if local

See Also: AX=FFBCh
