Int 21 Fn B802  - Novell Netware - Print Services - Get Specific Capture   [N]

   AX = B802h
   CX = size of reply buffer (01h-3Fh)
   DH = LPT port (00h-02h)
   ES:BX -> reply buffer for capture flags table (see #1493)

Return: AL = status
       00h successful

Note:  this function is supported by Advanced NetWare 2.1+

See Also: AX=B800h,AX=B803h,AH=DFh/DL=00h,AH=DFh/DL=04h
