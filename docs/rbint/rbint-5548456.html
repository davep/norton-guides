Int 68 Fn 01  - Appc/pc - Network Device Control - Display                 [N]

   AH = 01h subfn 1B00h
   DS:DX -> control block (see #3338)

Return: control block updated

See Also: AH=01h/SF=2000h,AH=01h/SF=2B00h,INT 68"Novell"

Format of APPC/PC "DISPLAY" control block:

 2Ah  8 BYTEs  (big-endian) mode name
