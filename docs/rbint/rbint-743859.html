Int 10 Fn FF  - Video - Oak Vga Bios V1.02+ - Set Emulation                [V]

   AH = FFh
   AL = emulation
       43h ('C') CGA emulation
       45h ('E') EGA emulation
       4Dh ('M') Hercules emulation
       56h ('V') VGA emulation
   ES:DI -> signature string "Calamity"

Return: VGA switched to suggested mode

See Also: AH=00h,AX=007Fh/BH=00h,AX=007Fh/BH=02h,AX=5F01h
