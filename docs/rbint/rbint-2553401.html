Int 21 Fn 5701  - Dos 2+ - Set File's Last-written Date And Time           [D]

   AX = 5701h
   BX = file handle
   CX = new time (see #1351)
   DX = new date (see #1352)

Return: CF clear if successful
   CF set on error
       AX = error code (01h,06h) (see #1366)

See Also: AX=5700h,AX=5705h"Windows95",AX=5707h"Windows95"
