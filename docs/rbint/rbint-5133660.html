Int 61 Fn 0007  - Banyan Vines - Verify Existence Of Name And Return Cano  [N]

   AX = 0007h
   BX = 0007h
   DS:DX -> NiceName block (see #3107)

Return: AX = status (see #3103)

See Also: AX=0007h/BX=0005h,AX=0007h/BX=0008h

Format of VINES NiceName block:
Offset Size    Description (Table 3107)
 00h   WORD    type of name
       0064h (100) organization
       00C8h (200) group
       012Ch (300) item
 02h   WORD    pointer to ASCIZ input name
 04h   WORD    pointer to 64-byte buffer for output name
