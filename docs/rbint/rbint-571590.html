Int 10 Fn 40  - Video - Hercules Grafix - "gmode" - Set Graphics Mode      [V]

   AH = 40h

Return: nothing
Desc:  switch the Hercules Graphics Card into graphics mode (720x348)

See Also: AH=00h,AH=41h
