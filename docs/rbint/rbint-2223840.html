Int 21 Fn 37DE  - Diet V1.43e - Read Ems Status                            [k]

   AX = 37DEh
   BX = 899Dh ('DI' + 'ET')

Return: AX = 0000h
   CX = EMS status
       0000h not used
       0001h used as work area
       0002h used for code and as work area
   DX = EMM handle when EMS is in use
