Int 94  u - Media Vision Pcm.com - Timer Control                           [s]

   AL = 10h
   SI = 8017h
   AH = timer options (see #3674)
   DX:CX = delay if AH bit 7 set (one-shot if DX bit 15 set)

Return: ???

See Also: AL=08h/SI=8017h

Bitfields for PCM.COM timer options:
Bit(s) Description (Table 3674)
 7 set timer
 6 timer active (timer turned off if clear)
 5-0   timer number
