Int 21 Fn 5E01  - Dos 3.1+ Network - Set Machine Name                      [D]

   AX = 5E01h
   CH = 00h undefine name (make it invalid)
      <> 0  define name
   CL = name number
   DS:DX -> 15-character blank-padded ASCIZ name

See Also: AX=5E00h
