Int 2F Fn 4A11  - Dblspace.bin V6.2 - Get Automount Drives                 [k]

   AX = 4A11h
   BX = 000Bh

Return: AX = 0000h if supported
       CX:DX = mask of drives with AutoMount enabled
       0000h:0000h on error
   BX destroyed

See Also: AX=4A11h/BX=000Ah
