Int 21 Fn 5706  - Ms-dos 7/windows95 - Get Creation Date And Time          [D]

   AX = 5706h
   BX = file handle

Return: CF clear if successful
       CX = creation time (see #1351)
       DX = creation date (see #1352)
       SI = number of 10-millisecond units past time in CX (0-199)
   CF set on error
       AX = error code

See Also: AX=5701h,AX=5704h"Windows95",AX=5707h
