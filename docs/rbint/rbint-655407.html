Int 10 Fn 5F01  - Chips & Technologies Extended Bios - Set Preprogrammed   [V]

   AX = 5F01h
   BL = emulation type (see #0115)

Return: AL = 5Fh if function supported
       AH = status
       00h unsuccessful
       01h successful

See Also: AX=5F00h,AX=5F02h,AH=FFh"Oak"

(Table 0115)
Values for Chips&Technologies emulation type:
 00h,01h reserved
 02h   CGA
 03h   MDA
 04h   Hercules
 05h   EGA
 06h   VGA
