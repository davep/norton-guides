Int 67 Fn 53  - Lim Ems 4.0 - Get/set Handle Name                          [m]

   AH = 53h
   AL = subfunction
       00h get handle name
          ES:DI -> 8-byte buffer for handle name
       01h set handle name
          DS:SI -> 8-byte handle name
   DX = handle

Return: AH = status (00h,80h,81h,83h,84h,8Fh,A1h) (see #3319)

See Also: AH=52h
