Int 61 Fn 0007  - Banyan Vines - Find A Service                            [N]

   AX = 0007h
   BX = 0003h
   DS:DX -> port control block (see #3105)

Return: AX = status (see AX=0007h/BX=0000h)

See Also: AX=0007h/BX=0002h,AX=0007h/BX=0004h,AX=0007h/BX=0005h

Format of port control block:
Offset Size    Description (Table 3105)
 00h   WORD    ???
 02h   WORD    ???
 04h 16    BYTEs   IPC port (see #2999 at AX=0001h"Sosock")
 14h  5    WORDs   ???
