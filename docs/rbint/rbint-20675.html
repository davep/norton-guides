ASCIZ

   A NUL-terminated ASCII string.  The ASCIZ string "ABC" consists
of the four bytes 41h, 42h, 43h, and 00h.  Unless otherwise specified,
maximum lengths given in the interrupt list do not include the
terminating NUL.
