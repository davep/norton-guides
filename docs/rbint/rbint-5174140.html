Int 61 Fn 18  - Pc/tcp Kernel V2.05+ - "net_eof" - Close Transmit Side Of  [N]

   AH = 18h
   BX = network descriptor

Return: CF clear if successful
   CF set on error
       AX = error code (see #2990 at INT 61"FTP Software")

Note:  a TCP "FIN" command is sent and no further data may be transmitted,
     although the connection remains open

See Also: INT 61"FTP Software",INT 61/AH=00h"PC/TCP",INT 61/AH=13h"PC/TCP"

See Also: INT 63/AH=0Eh"BW-TCP"
