Int 2F Fn 1403  - European Ms-dos 4.0 Popup - "restorepu" - Restore Scree  [D]

   AX = 1403h
   ES:DI -> buffer containing saved screen
       (0000h:0000h for default buffer in POPUP)

Return: CF clear if successful
   CF set on error
       AX = error code (see #2312)

See Also: AX=1400h"POPUP",AX=1401h"POPUP",AX=1402h"POPUP"
