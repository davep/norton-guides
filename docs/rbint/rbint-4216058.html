Int 2F Fn 9903  U - Dos Navigator Ii - Specify How To Execute Commands     [l]

   AX = 9903h
   CL = 1 to use INT 2E
   CL <> 1 to use INT 21/AH=4Bh

See Also: AX=9900h,AX=9901h,AX=9902h
