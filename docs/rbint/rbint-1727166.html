Int 1A Fn 04  - Time - Get Real-time Clock Date (at,xt286,ps)              [B]

   AH = 04h

Return: CF clear if successful
       CH = century (BCD)
       CL = year (BCD)
       DH = month (BCD)
       DL = day (BCD)
   CF set on error

See Also: AH=02h,AH=04h"Sperry",AH=05h,INT 21/AH=2Ah,INT 4B/AH=02h"TI"
