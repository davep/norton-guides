Int 6F Fn 1E  - Hp Vectra Ex-bios - "f_ram_get" - Get Ex-bios Free Ram Ar  [b]

   AH = 1Eh
   BP = 0012h (driver ID for V_SYSTEM)

Return: AH = status (00h) (see #3415 at INT 6F/AH=00h"HP Vectra")
   BX = last-used DS
   DX = maximum DS
   BP,DS destroyed
