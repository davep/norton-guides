Int 7C Fn 000F  - Sk-upps Data Link Interface Api - Feed Back Buffer       [N]

   AX = 000Fh
   ES:BX -> Mbuf (see #3518)

Return: nothing

Notes: A protocol may call this function instead of returning the Mbuf
     directly from the Receive Upcall if it is running in "promiscuous",
     "hook" or "chain" mode.

See Also: AX=0007h
