Int 1A Fn 02  - Tandy 2000 - Time - Get Date And Time                      [b]

   AH = 02h

Return: BX = number of days since January 1, 1980
   CH = hours
   CL = minutes
   DH = seconds
   DL = hundredths

See Also: AH=03h"Tandy 2000",INT 55"Tandy 2000"
