Int 21 Fn DF05  - Software Carousel - Change Partition Size                [T]

   AX = DF05h
   BL = task number
   DX = new size in KB

Return: AL = status
       00h Carousel not running
       01h successful
       FFh unsucessful
       AH = error code (01h,04h,05h) (see #1501)
   BX = minimum size allowed
   CX = maximum size available

See Also: AX=DF03h
