Int 10 Fn 0B  - Tandy 2000 - Video - Set Palette Entry                     [V]

   AH = 0Bh
   BH = 02h
   BL = palette entry number
   AL = new color value for palette entry

Return: nothing

Note:  this interrupt is identical to INT 52 for Tandy 2000

See Also: INT 10/AH=00h,INT 10/AH=0Bh/BH=01h,INT 52"Tandy 2000"
