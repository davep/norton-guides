Int 21 Fn 5FC9  - Lantastic V4.0+ - Cancel Server Shutdown                 [N]

   AX = 5FC9h
   ES:DI -> ASCIZ server name in form "\\machine"

Return: CF clear if successful
   CF set on error
       AX = error code

Note:  you must have the "S" privilege to use this call

See Also: AX=5FC8h
