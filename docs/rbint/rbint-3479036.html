Int 2A Fn 01  - Network (microsoft,lantastic) - Execute Netbios Request,n  [N]

   AH = 01h
   ES:BX -> NCB (see #2920 at INT 5C"NetBIOS")

Return: AL = NetBIOS error code
   AH = status
       00h no error
       01h error occurred

See Also: AH=04h,AX=0500h,INT 5B"Extended NetBIOS",INT 5C"NetBIOS"
