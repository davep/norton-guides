Int 14 Fn E8  - Telapi - "telattach" - Attach Com Port To/from Telnet Ses  [N]

   AH = E8h
   BX = connection ID (see AH=E0h"TelAPI")
   CX = serial port number (0000h-0003h = COM1-COM4)

Return: AX = status
       0000h successful
       FFFFh failed

See Also: AH=E0h,AH=E9h,AX=FF00h
