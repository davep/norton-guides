Int 63 Fn 00  - Hp 100lx - Map High Memory                                 [b]

   AH = 00h
   AL = physical page (00h seg C000, 01h seg C400h, ...)
   BX = zero-based logical page
   CX = page number
   DX = device ID (00h system ROM, 05h plugin, etc.)
