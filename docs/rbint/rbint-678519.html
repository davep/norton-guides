Int 10 Fn 73  - Hunter 16 - Select Text Font                               [V]

   AH = 73h
   AL = new font
         00h  7x7 in a 8x8 cell
         01h  5x7 in a 6x8 cell
         06h  3x5 in a 4x6 cell

Return: BH = cell width
   BL = cell height

See Also: AH=60h"HUNTER",AH=77h"HUNTER"
