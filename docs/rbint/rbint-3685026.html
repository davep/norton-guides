Int 2F Fn 1404  CU - Nlsfunc.com - Get Country Info                        [U]

   AX = 1404h
   BX = code page (see #1446 at INT 21/AX=6602h)
   DX = country code (see #1088 at INT 21/AH=38h)
   DS:SI -> internal code page structure (see #2305)
   ES:DI -> user buffer

Return: AL = status
        ???

Notes: this function is called by the DOS v3.3+ kernel on INT 21/AH=38h
   code page structure apparently only needed for COUNTRY.SYS pathname

See Also: AX=1402h,AX=1403h,INT 21/AH=38h"GET"
