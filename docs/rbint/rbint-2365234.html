Int 21 Fn 4403  U - Qualitas 386max V6.01+ - Turn 386max On                [m]

   AX = 4403h subfn 02h
   BX = handle for device "386MAX$$"
   DS:DX -> BYTE 02h
   CX ignored

Return: DS:DX -> BYTE status (00h = successful)
   CF clear if successful
       AX destroyed
   CF set on error
       AX = error code (01h,05h,06h,0Dh) (see #1366 at AH=59h/BX=0000h)

See Also: AX=4403h/SF=01h,AX=4403h/SF=03h,AX=4403h/SF=05h
