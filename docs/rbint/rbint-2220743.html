Int 21 Fn 37D2  - Diet V1.43e - Get Tsr Control Flags                      [k]

   AX = 37D2h
   BX = 899Dh ('DI' + 'ET')

Return: AX = 0000h
   DL = control flag (00h active, else disabled)
   DH = skip flag (nonzero while TSR active)

See Also: AX=37D0h,AX=37D3h,AX=37D4h
