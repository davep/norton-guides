Int F7 Fn 01  - Fsbbs 2.0 - User Record                                    [-]

   AH = 01h
   AL = function
       00h get user record for user currently online

       Return: DS:DX -> user record
       01h set user record
       DS:DX -> user record

       Return: nothing

See Also: AH=00h,AH=02h
