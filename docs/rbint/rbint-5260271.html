Int 62 Fn 003A  u - Fgdriver V4.02 - "fg_getattr" - Get Character Attrib   [V]

   AX = 003Ah
   BX = row
   CX = column

Return: AX = character attribute at specified location on active video page

Notes: the FGDRIVER functions are rearranged with each major release, but
     their parameters do not change (see #3145 at AX=0000h)
   This call is ignored in graphics modes

See Also: AX=000Bh,AX=003Dh,AX=0040h,AX=00A6h
