Int 2F Fn BC06  - Mediavision Mvsound.sys - Get Status String              [s]

   AX = BC06h

Return: AX = 4D56h ('MV')
   DX:BX -> status string (first byte 0Ch if no status message to display)

See Also: AX=BC00h/BX=3F3Fh,AX=BC01h"MVSOUND",AX=BC0Bh"MVSOUND"
