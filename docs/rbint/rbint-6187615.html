Int E1  - Mp/m-86, - Alternate Cp/m-86 Function Calls                      [O]

   CL = function number (see #3690,#3691)
   DS,DX = parameters

Return: as appropriate for function
   CX is often the error code (see #3692)
Desc:  used by some applications which alter CP/M functions while running a
     child program, to store the original INT E0 vector before
     intercepting INT E0

See Also: #3690 at INT E0"CP/M"
