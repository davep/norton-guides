Int 16 Fn 22  - Hunter 16 - Control Ctrl-alt-del                           [b]

   AH = 22h
   AL = new Ctrl-Alt-Del state (00h enabled, nonzero disabled)
   BX = 0708h
   CX = 0910h
   DX = 1112h

Return: AL = 00h if successful

See Also: AH=21h"HUNTER",AH=23h"HUNTER",AH=2Ah
