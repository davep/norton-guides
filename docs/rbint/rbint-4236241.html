Int 2F Fn AC10  - Qrip/tsr - Set Output Color For Protected Area           [V]

   AX = AC10h
   DI = 1092h
   BL = new attribute (bit 7 set for high-intensity background)

Return: nothing

Note:  if DI <> 1092h on entry, QRIP chains the call

See Also: AX=AC0Dh,AX=AC0Eh,AX=AC0Fh,AX=AC11h,AX=ACF0h
