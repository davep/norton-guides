Int 2F Fn DB11  - Zyxel Zfax V3+ - Send Fax                                [F]

   AX = DB11h
   DS:SI -> filename including path (max 128 characters)
   DS:BX -> remote FAX number

Return: AX = status (see #2768)

See Also: AX=DB00h"ZFAX",AX=DB12h,AX=DB13h,AX=DB14h,AX=DB15h,AX=DB20h,AX=DB21h
