Int 32 Fn 6E00  - Noise.sys V0.53+ - Installation Check                    [y]

   AX = 6E00h

Return: AL = installation status
       00h not installed
       FFh installed
       CX = version (ie, 0123h = Version 1.2.3)
       DX:DI -> signature string
