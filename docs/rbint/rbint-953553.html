Int 14 Fn 16  - Digiboard Digichannel Pc/x* - Ccb Command                  [S]

   AH = 16h
   AL = CCB command number (see #0268) (see also following entries)
   BL = byte 2
   BH = byte 3
   CL = byte 1 (for all channel functions except 4Eh and 4Fh)
   DX = port number (00h-03h) (04h-43h for XAPCM232.SYS)

Return: AH = status
       00h successful
       80h error
       FFh error

See Also: AX=1646h,AH=18h"Digiboard"
