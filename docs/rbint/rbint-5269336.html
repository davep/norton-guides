Int 62 Fn 004A  u - Fgdriver V4.02 - "fg_getmaxy" - Get Maximum Row In Sc  [V]

   AX = 004Ah

Return: AX = maximum Y coordinate in screen space
       (or character space if in text mode)

Note:  the FGDRIVER functions are rearranged with each major release, but
     their parameters do not change (see #3145 at AX=0000h)

See Also: AX=0049h,AX=00C1h
