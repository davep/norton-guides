Int 61 Fn 0F  - Atari Portfolio - Get/set Cursor Mode                      [b]

   AH = 0Fh
   AL = function
       00h get mode
       01h set mode
       BL = new cursor mode
           00h cursor off
           01h underline
           02h block
       02h force mode (set size to reflect NumLock state)

Return: BL = current/old cursor mode

See Also: AH=00h"Portfolio",AH=0Eh"Portfolio",AH=10h"Portfolio",INT 10/AH=01h
