Int 2C Fn 0007  P - Cloaking - Set Protected-mode Interrupt Vector         [m]

   AX = 0007h
   CL = vector (00h-7Fh)
   DX:EBX -> interrupt handler

Return: CF clear

Notes: this function was first introduced with RM386 (RAM-MAN/386) v6.00
   the IDT entry's type remains unchanged

See Also: AX=0006h,INT 31/AX=0205h
