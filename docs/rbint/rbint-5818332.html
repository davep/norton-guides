Int 7C Fn 0000  - Sk-upps Data Link Interface Api - Get Driver Version     [N]

   AX = 0000h

Return: AH = major DLI version
   AL = minor DLI version
   DX = OEM signature (see #3516)
   CL = DLI 2.8+ topology: 1=Ethernet, 2=FDDI, 3=Token-Ring

See Also: AX=0002h,AX=000Bh,AX=000Dh,AX=000Eh
