Int 2F Fn AB02  - Btrieve Multi-user - Get New Process Id                  [f]

   AX = AB02h

Return: AL = 00h successful
       BX = process ID
   AL > 00h failed, retry after calling INT 7F/AX=0200h

See Also: AX=AB00h,AX=AB01h,INT 7B"Btrieve",INT 7F/AX=0200h
