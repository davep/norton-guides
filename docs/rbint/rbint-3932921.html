Int 2F Fn 4010  - Os/2 V2.0+ - Installation Check / Get Version            [O]

   AX = 4010h

Return: AX = 4010h if OS/2 not installed
   AX = 0000h for OS/2 Warp 3.0
   BX = OS/2 version if installed

Note:  OS/2 Warp 3.0

See Also: INT 21/AH=30h,INT 21/AX=3306h
