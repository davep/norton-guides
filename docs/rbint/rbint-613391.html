Int 10 Fn 4F10  - Vesa Vbe/pm (power Management) V1.0+ - Set Display Powe  [p]

   AX = 4F10h
   BL = 01h
   BH = new state (see #0088)

Return: AL = 4Fh if function supported
   AH = call status
       00h successful
       else failed

See Also: AX=4F10h/BL=00h,AX=4F10h/BL=02h,AX=A00Ch

(Table 0088)
Values for VESA VBE/PM power state:
 00h   On
 01h   standby
 02h   suspend
 04h   Off
 08h   reduced On (for flat screens)
