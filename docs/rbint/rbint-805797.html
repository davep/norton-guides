Int 13 Fn 15  - Disk - Get Disk Type (xt 1/10/86 Or Later,xt286,at,ps)     [B]

   AH = 15h
   DL = drive number (bit 7 set for hard disk)

Return: CF clear if successful
       AH = type code
       00h no such drive
       01h floppy without change-line support
       02h floppy (or other removable drive) with change-line support
       03h hard disk
           CX:DX = number of 512-byte sectors
   CF set on error
       AH = status (see #0166 at AH=01h)

Note:  SyQuest can report type 01h or 02h for 'hard disks', since its media
     is removable

See Also: AH=08h,AH=16h,AH=17h,AH=19h"SCSI"
