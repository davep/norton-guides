Int 6F Fn 0C  - Hp Vectra Ex-bios - "f_get_block" - Read Blocks Of Data F  [b]

   AH = 0Ch
   BP = driver ID (see #3412)
   DH = device major number
   DL = device minor number
   ES:DI -> command block (see #3437)

Return: AH = status (see #3415 at AH=00h"HP Vectra")
   BX = operation status
   BP,DS destroyed

See Also: AH=08h"F_GET_BYTE",AH=0Ah"F_PUT_BLOCK",AH=0Ch"F_GET_BUFFER"
