Int 15 Fn 8401  - Hunter 16 - Set Diskette Port                            [b]

   AX = 8401h
   BH = Drive (0: A:, 1: B:)
   BL = Port (0: COM1, >0: COM2)
Desc:  set the COM port used for the floppy drive

See Also: AX=8400h
