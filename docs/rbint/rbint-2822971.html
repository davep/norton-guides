Int 21 Fn C501  - Novell Netware - Synchronization Services - Examine Sem  [N]

   AX = C501h
   CX:DX = semaphore handle

Return: AL = status
       00h successful
       CX = semaphore value (-127 to 127)
       DL = count of processes which have the semaphore open
       FFh invalid handle

Note:  this function is supported by NetWare 4.6+, Advanced NetWare 1.0+, and
     Alloy NTNX

See Also: AX=C500h"NetWare",AX=C502h,AX=C504h,AX=F220h/SF=01h
