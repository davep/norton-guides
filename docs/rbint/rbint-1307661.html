Int 15 Fn 61  U - Hp 100lx/200lx - Set Announciators Position              [b]

   AH = 61h
   AL = position (20h = left, 60h = right)

Note:  The announciators are the indicator symbols for the Shift and Fn keys
     in the bottom line

See Also: AH=62h"HP",INT 16/AH=02h
