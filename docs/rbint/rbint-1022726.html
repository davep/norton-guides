Int 14 Fn 82  - Couriers.com - Configure Port                              [S]

   AH = 82h
   AL = port number (1-4)
   BX = speed (bps)
   CX = bit flags
       bit 0: enable input flow control
       bit 1: enable output flow control
       bit 2: use X.PC protocol (not yet implemented)

Return: nothing

See Also: AH=00h,AH=8Ch,INT 7A"X.PC"
