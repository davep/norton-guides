Int 18 Fn 6909  - Gravis Ultra Sound Yea_gus.exe - Ramp Volume             [s]

   AX = 6909h
   BL = voice number (00h-1Fh)
   BH = looping type (0 = none, 1 = forward, 2 = back and forth)
   CX = starting volume
   DX = ending volume
   DI:SI = time

Return: nothing

See Also: AX=6902h,AX=690Ah
