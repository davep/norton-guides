Int 2C Fn 0002  P - Cloaking - Set Segment Base Address                    [m]

   AX = 0002h
   SI = selector
   EBX = new physical base addres

Return: CF clear if successful
   CF set on error
       AX = error code (see #2244)

Note:  this function was first introduced with RM386 (RAM-MAN/386) v6.00

See Also: AX=0000h,AX=0003h,AX=0004h,INT 31/AX=0007h,#0427
