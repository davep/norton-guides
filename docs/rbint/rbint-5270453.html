Int 62 Fn 004D  u - Fgdriver V4.02 - "fg_getpixel" - Get Color Of Specifi  [V]

   AX = 004Dh
   BX = column in screen space
   CX = row in screen space

Return: AX = color value of pixel (0 to num_colors-1)
       0000h in text modes

Note:  the FGDRIVER functions are rearranged with each major release, but
     their parameters do not change (see #3145 at AX=0000h)

See Also: AX=003Ah,AX=003Dh,AX=008Fh,AX=0090h
