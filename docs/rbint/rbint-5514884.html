Int 67 Fn 5A  - Lim Ems 4.0 - Allocate Standard/raw Pages                  [m]

   AH = 5Ah
   AL = subfunction
       00h allocate standard pages
       01h allocate raw pages
   BX = number of pages to allocate

Return: DX = handle
   AH = status (00h,80h,81h,84h,85h,87h,88h,8Fh) (see #3319)
