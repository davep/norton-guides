Int 69 Fn 0105  - Decnet Dos Cterm - Open Session                          [N]

   AX = 0105h
   DS:BX -> ASCIZ node name
   ES:DX -> buffer for session control block (see #3398 at INT 6A/AH=D0h)

Return: AX <= 0 on error
   AX > 0 session handle

See Also: AX=0103h,AX=0106h,AX=010Ah
