Int 16 Fn Sect  - Hp Vectra Ex-bios - "f16_inquire" - Extended Bios Insta  [!]

   AX = 6F00h
   BX <> 4850h (usually set to 0000h for simplicity)

Return: BX = 4850h ("HP") if present

Notes: called by recent MS Mouse drivers looking for an HP-HIL mouse
   supported by the original HP Vectra AT and ES/QS/RS series HP Vectras

See Also: AX=6F01h,AX=6F04h,AX=6F07h,AX=6F09h,AX=6F0Dh,AX=6F11h,AX=6F12h

See Also: INT 14/AX=6F00h,INT 17/AX=6F00h,INT 33/AX=6F00h

See Also: INT 6F/AH=00h"HP Vectra"
