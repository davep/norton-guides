Int 14 Fn A5  - 3com Bapi Serial I/o - Read Block                          [S]

   AH = A5h
   CX = length of buffer
   DH = session ID (00h for external session managment)
   ES:BX -> buffer for data

Return: AH = return code (00h,02h,03h,07h,0Ah,0Bh) (see #0320)
   CX = number of bytes actually read

See Also: AH=18h,AH=83h"COURIERS",AH=A0h"BAPI",AH=A3h"BAPI",AH=A4h"BAPI"

See Also: AH=A7h"BAPI",AX=FF02h
