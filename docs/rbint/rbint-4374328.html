Int 2F Fn CB06  - Communicating Applications Specification - Find Next Qu  [F]

   AX = CB06h
   DL = queue to search
       00h task queue
       01h receive queue
       02h log queue

Return: AX = 0000h successful
       BX = event handle for found event
   AX < 0     error code (see #2720)

Note:  direction of search is same as preceding FIND FIRST call

See Also: AX=CB05h
