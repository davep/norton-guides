Int 6F Fn 0E  - Hp Vectra Ex-bios - "f_set_limits_y" - Set Vertical Track  [b]

   AH = 0Eh
   BP = 005Ah (driver ID for V_STRACK)
   CX = minimum Y coordinate
   DX = maximum Y coordinate

Return: AH = status (see #3415)
   BP,DS destroyed

See Also: AH=00h"HP Vectra",AH=04h/BP=005Ah,AH=08h/BP=005Ah,AH=0Ch/BP=005Ah
