Int 14 Fn 4003  - I1541 - Reset All Devices                                [t]

   AX = 4003h

Return: nothing
Desc:  send a reset pulse of 100ms to all CBM devices

Note:  it is necessary to wait about 2 seconds after reset before executing
     other instructions

See Also: AX=4000h,AX=4004h
