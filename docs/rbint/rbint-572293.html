Int 10 Fn 43  - Video - Hercules Grafix - "gpage" - Select Drawing Page    [V]

   AH = 43h
   AL = page number (0,1)

Return: nothing
Desc:  specify which of the two pages of video memory is to be used for
     output

See Also: AH=05h,AH=44h,AH=45h
