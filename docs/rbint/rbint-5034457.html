Int 60 Fn 15  - Ftp Packet Driver - Extended Func - Get Receive Mode       [N]

   AH = 15h
   BX = handle (optional for v1.10+)

Return: CF set on error
       DH = error code (01h) (see #2976)
   CF clear if successful
       AX = receive mode (see #2986)

See Also: AH=14h
