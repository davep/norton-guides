Int 1A Fn FF02  - Snd - Start Sound                                        [s]

   AX = FF02h
   DX = frequency in Hertz (14h-FFFFh)

Return: AH = status
       00h successful
       01h SND busy

See Also: AX=FF00h,AX=FF01h,AX=FF03h
