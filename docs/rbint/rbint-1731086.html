Int 1A Fn 0E  - Time - Get Rtc Date/time Alarm And Status (ibm)            [B]

   AH = 0Eh

Return: CF clear if successful
       BH = alarm status
       00h disabled
       01h enabled but will not power-up system
       02h enabled, system will power-up on activation
       CH = alarm time, hours (BCD)
       CL = alarm time, minutes (BCD)
       DH = seconds (BCD)
       DL = day of month (BCD)
   CF set on error

See Also: AH=0Ch,AH=0Dh,AH=0Fh
