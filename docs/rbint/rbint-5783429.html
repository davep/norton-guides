Int 7A  - Novell Netware - Spx Driver - Listen For Spx Packet              [N]

   BX = 0017h
   DX = connection ID (unused in v2.15)
   ES:SI -> Event Control Block (see BX=0003h)

Notes: this function is supported by Advanced NetWare 2.1+
