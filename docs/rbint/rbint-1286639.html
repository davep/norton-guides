Int 15 Fn 5380  - Apm Sl Enhanced V1.0 - Set Calendar Event Date           [p]

   AX = 5380h
   BH = 0Bh
   SI = century
   DI = year
   CH = month
   CL = day

Return: CF clear if successful
   CF set on error
       AH = error code (see #0409)
Desc:  sets calendar event date in Extended CMOS ram

See Also: AX=5380h/BH=0Ah,AX=5380h/BH=7Fh
