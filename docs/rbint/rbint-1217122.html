Int 15 Fn 1C  - Multidos Plus - Release Resource Semaphore By Name         [T]

   AH = 1Ch
   DS:SI -> 8-byte name

Return: AH = status
       00h successful
       01h not semaphore owner
       02h invalid semaphore number
       04h out of string space

Notes: (see AH=02h"MultiDOS")
   equivalent to AH=1Ah followed by AH=02h

See Also: AH=02h"MultiDOS",AH=1Ah"MultiDOS",AH=1Bh"MultiDOS",AH=1Dh"MultiDOS"
