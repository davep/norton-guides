Int 2F Fn CBDD  - Comwave Microfax Specification - Command Submission      [F]

   AX = CBDDh
   BL = 56h
   BH = ?
   CX:DX -> command to execute

Return: AX = status
       0000h success
       FFFFh on error

See Also: AH=2Ah,AX=CBDCh,AX=80FBh
