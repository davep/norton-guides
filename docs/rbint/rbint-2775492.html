Int 21 Fn 7304  - Windows95 - Fat32 - Set Dpb To Use For Formatting        [D]

   AX = 7304h
   DL = drive number (00h=default, 01h=A:, etc.)
   ES:DI -> buffer for Set_DPBforFormat structure (see #1478)

Return: CF clear if successful
       ES:DI buffer updated
   CF set on error
       AX = error code

See Also: AX=7302h,AX=7303h,AX=7305h

Format of Set_DPBforFormat structure:
