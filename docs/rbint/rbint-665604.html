Int 10 Fn 6F00  - Video - Video7 Vga,vega Vga - Installation Check         [V]

   AX = 6F00h
   BX = 0000h

Return: BX = 5637h ('V7') indicates Video7 VGA/VEGA VGA extensions are present

See Also: AX=6F01h,AX=6F02h,AX=6F03h,AX=6F04h,AX=6F05h
