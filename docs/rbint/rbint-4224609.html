Int 2F Fn AB01  - Srsoft Modal Pc V2+ - Get Process Run Parameter          [-]

   AX = AB01h

Return: AX = 5253h ('SR' byte-swapped) if installed
   BL = run parameter
       00h normal process
       01h run with tracing enabled
       02h run initialization process (first thread of main task)

See Also: AX=AB00h"MODAL",AX=AB02h"MODAL"
