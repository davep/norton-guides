Int 2F Fn CB0C  - Communicating Applications Specification - Get Event Ti  [F]

   AX = CB0Ch
   BX = event handle
   DL = queue
       00h task queue
       01h receive queue
       02h log queue

Return: AX = 0000h successful
       CH = hour
       CL = minute
       DH = second
       DL = 00h
   AX < 0     error code (see #2720)

See Also: AX=CB0Ah,AX=CB0Dh
