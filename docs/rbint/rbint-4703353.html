Int 33 Fn 0024  - Ms Mouse V6.26+ - Get Software Version, Mouse Type, And  [M]

   AX = 0024h

Return: AX = FFFFh on error
   otherwise,
       BH = major version
       BL = minor version
       CH = type (1=bus, 2=serial, 3=InPort, 4=PS/2, 5=HP)
       CL = interrupt (0=PS/2, 2=IRQ2, 3=IRQ3,...,7=IRQ7,...,0Fh=IRQ15)

See Also: AX=004Dh,AX=006Dh
