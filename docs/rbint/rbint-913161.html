Int 14 Fn 04  - Serial - Extended Initialize (convertible,ps)              [S]

   AH = 04h
   AL = break status
       00h if break
       01h if no break
   BH = parity (see #0233)
   BL = number of stop bits
       00h one stop bit
       01h two stop bits (1.5 if 5 bit word length)
   CH = word length (see #0234)
   CL = bps rate (see #0235)
   DX = port number

Return: AX = port status code (see #0230,#0231)

See Also: AH=00h,AH=1Eh,AX=8000h"ARTICOM"
