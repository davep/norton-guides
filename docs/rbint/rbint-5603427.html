Int 6A Fn D400  - Decnet Dos Local Area Transport - Copy Lat Counters      [N]

   AX = D400h
   DH = FFh
   CX = buffer size
   ES:BX -> buffer for LAT counters

Return: AX = status
       0000h counters copied into buffer
       FFFFh buffer too small

See Also: AX=D300h
