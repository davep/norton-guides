Int 2F Fn E201  - Securedevice - Get Information                           [y]

   AX = E201h
   DX = driver index (0000h = first loaded)

Return: AL = number of volumes handled by driver
   DL = drive number of first volume (00h = A:)

See Also: AX=E200h,AX=E203h,AX=E209h
