Int E0  - Conccp/m,dr Multiuser Dos - "c_attach" - Attach To Default Cons  [O]

   CL = 92h

Return: AX = status

Notes: also supported by REAL/32
   if the console is currently owned by another process, this function
     waits until the console is available

See Also: INT E0/CL=93h,INT E0/CL=94h,INT E0/CL=99h,INT E0/CL=9Eh,INT E0/CL=A2h
