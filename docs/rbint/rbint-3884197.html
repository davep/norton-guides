Int 2F Fn 1AAB  - Ansiplus V3.01+ - Set Ansiplus Key Repeat Rate           [V]

   AX = 1AABh
   BX = repeat rate in characters per second
       0000h use BIOS repeat rate

Return: nothing

See Also: AX=1A00h/BX=414Eh,AX=1AAAh,AX=1AACh
