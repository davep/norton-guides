Int 21 Fn B8  - Attachmate Extra! - Disable Host Buffer Updates            [I]

   AH = B8h
   AL = window number (1-4=host B-E)
   DL = 01h

Notes: only valid in CUT mode
   next AID keystroke (eg Enter) enables host buffer updates
