Int 1A Fn 7400  - Snap.exe 3.2+ - "snap_version" - Get Version             [N]

   AX = 7400h

Return: AX = version (AH=major, AL=minor)

Note:  this call is only valid if SNAP is installed

See Also: AX=7302h,INT 1A/AX=6F01h
