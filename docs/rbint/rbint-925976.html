Int 14 Fn 05  - Multidos Plus Iodrv - Read Character From Port             [S]

   AH = 05h
   AL = timeout in seconds (00h = never)

Return: AL = status
       00h successful
       AH = character read
       01h read error
       02h timed out
       other modem status (CTS, DSR) changed

Note:  the port number is stored at offset BEh in the Task Control Block

See Also: AH=02h,AH=04h"MultiDOS",AH=06h"MultiDOS",AH=22h"MultiDOS"
