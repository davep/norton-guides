Int B3 Fn 7F  - Zipkey - Enable/disable Hotkeys                            [U]

   AH = 7Fh
   BL = function
       00h turn off hotkeys
       01h turn on hotkeys
       02h return hotkey status
       03h toggle hotkey status

Return: AL = hotkey status
       00h off
       01h on
