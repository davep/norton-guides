Int 61 Fn 0007  - Banyan Vines - Get Mm Port For A Service                 [N]

   AX = 0007h
   BX = 0001h
   DS:DX -> StreetTalk service name (64 bytes)
   DS:DI -> 16-byte buffer for IPC port (see #2999 at AX=0001h"Sosock")

Return: AX = status (see #3103)

Notes: This function calls AX=0007/BX=0002h with the port count set to 1 and
     the port type set to 00h

See Also: AX=0007h/BX=0002h,AX=0007h/BX=0004h,AX=0007h/BX=000Ah
