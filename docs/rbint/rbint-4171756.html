       0000h 279 mm (11 in)
       0001h 297 mm (11.69 in)

       0000h 215 mm (8.46 in)
       0001h 255 mm (10.04 in)
       0002h 303 mm (11.93 in)
       0003h 151 mm (5.91 in)
       0004h 107 mm (4.21 in)

       0000h only as FAX
       0001h only as file (for non-faxable files)
