Int 67 Fn 5BF4  - Rm386 V6.00 - Get Rm386 Global Flags                     [m]

   AX = 5BF4h

Return: AH = 00h
   BX = global flags 1 (see #3330)
   CX = global flags 2 (see #3331)
   DX = global flags 3 (see #3332)
   SI = global flags 4 (see #3333)

See Also: AX=5BF0h
