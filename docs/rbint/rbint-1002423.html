Int 14 Fn 8002  - Comm-drv V14.0 - Read Port Metrics - Get Port Parameter  [S]

   AX = 8002h
   DX = port number

Return: AH bit 7 set on error
   AH bit 7 clear if successful
       BX:DI -> Port Control Block (see #0293) (modifyable portion only)
