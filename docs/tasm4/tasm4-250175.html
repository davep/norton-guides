Pointer expression needs brackets

In Ideal mode, the operand contained a memory-referencing symbol that was
not surrounded by brackets to indicate that it references a memory
location. For example,

   B  DB   0 
      mov  al,B   ;warning, Ideal mode needs [B] 

Since MASM mode does not require the brackets, this is only a warning.
