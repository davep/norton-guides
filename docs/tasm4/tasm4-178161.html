/s        Sequential segment ordering (default)

          The /s option tells Turbo Assembler to place segments in the
          object file in the order in which they were encountered in the
          source file.

          By default, Turbo Assembler uses sequential segment-ordering;
          it can be changed by placing an /a option in the configuration
          file.
