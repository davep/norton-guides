WARN             Enable warning(s)                                  Directive

  WARN [warnclass]

    Enables the type of warnings specified with warnclass, or all warnings
    if warnclass is not specified.

          ALN     Segment alignment
          ASS     Assuming segment is 16-bit
          BRK     Brackets needed
          GTP     Global type doesn't match symbol type
          ICG     Insufficient code generation
          INT     INT 3 generation
          LCO     Location counter overflow
          MCP     MASM compatibility pass
          OPI     Open IF conditional
          OPP     Open procedure
          OPS     Open segment
          OVF     Arithmetic overflow
          PDC     Pass-dependent construction encountered
          PQK     Assuming constant for [const]
          PRO     Write-to-memory in protected mode using CS override
          RES     Reserved word warning
          TPI     Turbo Pascal illegal warning
          UNI     For turning off unitialized segment warning (cf.
                  segment combine type UNINIT)


    By default, TASM starts assembling with all warnings enabled except
    warn classes ICG and PRO.
