%TITLE           Specify listing title                              Directive

  %TITLE "text"

    Sets title in listing file to text. The title text appears at the top
    of each page, after the name of the source file and before any
    subtitle set with the %SUBTTL directive. You can use %TITLE as many
    times as you want.
