LABEL            Define symbol                                      Directive

  LABEL name complex_type               ; Ideal mode
  name LABEL complex_type               ; MASM mode

    Defines a symbol with a specified type. name must not have been
    defined previously in the source file. complex_type describes the
    size of the symbol and whether it refers to code or data.
