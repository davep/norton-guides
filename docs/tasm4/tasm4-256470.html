Too many initial values

You have supplied too many values in a structure or union initialization.
For example,

   STRUC XYZ
   A1 DB ? 
   A2 DD ? 
   ENDS  XYZ
   ANXYZ XYZ   <1,2,3> ;error, only 2 members in XYZ

You can supply fewer initializers than there are members in a structure or
union, but never more.
