%CREFALL         List all symbols                                   Directive

  %CREFALL

    Causes all subsequent symbols in the source file to appear in the
    cross-reference listing. This is the default mode for TASM.
    %CREFALL reverses the effect of any previous %CREFREF or %CREFUREF
    directives that disabled the listing of unreferenced or referenced
    symbols.
