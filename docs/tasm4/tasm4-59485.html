LARGESTACK       Indicates 32-bit stack                             Directive

  LARGESTACK

    Indicates that the stack is 32 bit.

    When 80386 code generation is enabled, the stack segment can be either
    16-bit or 32-bit. To generate the correct procedure prolog and epilog
    code in a program that does not have a MODEL statement, Turbo
    Assembler must know the stack size attribute.
