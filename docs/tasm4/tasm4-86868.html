QUIRKS           Enable TASM to use MASM "bugs"                     Directive

  QUIRKS                                ; Superseded by VERSION

    This directive was included in TASM to make it more compatible with
    MASM. If you are having trouble transferring source code from MASM to
    TASM, you may want to include this directive.

    See the TASM documentation for a full description of QUIRKS.
