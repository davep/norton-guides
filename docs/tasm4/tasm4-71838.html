NOEMUL           Disable coprocessor emulation                      Directive

  NOEMUL

    Causes all subsequent numeric coprocessor instructions to be generated
    as real instructions, instead of emulated instructions. When your
    program is executed, you must have an 80x87 coprocessor installed or
    these instructions will not work properly.

    This is the default floating-point assembly mode for Turbo Assembler.
