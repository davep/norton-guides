Turbo Link  Version 1.00 for Win32 Copyright (c) 1993 Borland International
Syntax: TLINK objfiles, exefile, mapfile, libfiles, deffile, resfiles
@xxxx indicates use response file xxxx

  -m      Map file with publics     -x      No map file at all
  -s      Detailed map of segments  -L      Specify library search paths
  -c      Case significant link     -v      Full symbolic debug information
  -Enn    Max number of errors      -n      No default libraries
  -B:xxxx Specify image base addr   -S:xxxx Specify application stack size
  -A:dd   Specify file alignment in power of 2
  -wxxx   Warning control
  -Txx    Specify output file type
             -Tpx  PE image
                (third letter can be e=EXE, d=DLL)
  -ax     Specify application type
             -ap   Windowing Compatible
             -aa   Uses Windowing API
  -Gx     Goodies
             -Gm   Write mangled names in map file
