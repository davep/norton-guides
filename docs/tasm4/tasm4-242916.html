name must come first

You put a symbol name after a directive, and the symbol name should come
first. For example,

   STRUC ABC    ;error, ABC must come before STRUC 

Since Ideal mode expects the name to come after the directive, you will
encounter this error if you try to assemble Ideal mode programs in MASM
mode.
