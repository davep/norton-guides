Expecting offset or pointer quantity

An expression expected an operand that referred to an offset within a
specific segment, but did not encounter the right sort of operand. For
example,

   SEGMENT CODE
      mov  ax,SEG CODE                 ;error, code is a segment not
                                        ; a location within a segment
   ENDS CODE
