FARDATA          Define far data segment                            Directive

  FARDATA [segname]                     ; Ideal or MASM mode
  .FARDATA [segname]                    ; MASM mode

    Defines the beginning of an initialized far data segment in a module.
    You must first have specified a memory model with the MODEL directive.
    You can have more than one far initialized data segment per module.

    If you don't specify a name, TASM uses segment name FAR_DATA.
