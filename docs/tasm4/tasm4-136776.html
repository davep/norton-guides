DUP              Duplicate occurrences                               Operator

  count DUP (expression [,expression] ...)

    Repeats a data allocation operation count times. count cannot be
    relative or forward-referenced.

    Examples:   var1    db      8 DUP (" ")
                var2    dw      2 DUP (3 DUP (4))
                var3    astruc  200 DUP (?)
