Unmatched ENDS: __

The ENDS directive has a name that does not match either the SEGMENT
directive that opened a segment or the STRUC or UNION directive that
started a structure or union definition. For example,

   STRUC ABC
   ENDS XYZ      ;error, XYZ should be ABC
   SEGMENT DATA
   ENDS CODE     ;error, code should be DATA
