Resource Linker  Version 2.00 Copyright (c)  1992, 1993 Borland International.

Syntax is: RLINK [options] [.RES input file] <.EXE input/output file>

        options marked with a '*' are on by default

        @<filename>     Take instructions from command file
        -d      Remove resources from EXE (no res file specified)
        -h      Display this help
        -?      Display this help
        -fe <filename>  Rename output .EXE file
        -fi <filename>  Specify additional input files
        -k      Do not re-order segments for fastload
        -t      Ignored for compatibility
        -30     Ignored for compatibility
        -31     Ignored for compatibility
        -v      Run in verbose mode
        -vx     Verbose mode with no EXE file (list resources)
