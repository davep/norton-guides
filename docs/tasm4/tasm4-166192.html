/c        Enable cross-reference in listing file

          Turbo Assembler adds the cross-reference information to the
          symbol table at the end of the listing file. This means that, in
          order to see the cross-reference information, you must either
          explicitly specify a listing file on the command line or use the
          /l option to enable the listing file.

          For each symbol, the cross-reference shows the line on which it
          is defined and all lines that refer to it.
