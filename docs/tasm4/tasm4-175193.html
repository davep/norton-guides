/n        Suppresses symbol table in listing file

          The /n option indicates that you don't want the usual symbol
          table at the end of the listing file. Normally, a complete
          symbol table listing appears at the end of the file, showing all
          symbols, their types, and their values.

          You must specify a listing file, either explicitly on the command
          line or by using the /l option; otherwise, /n has no effect.
