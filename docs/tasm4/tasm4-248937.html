Open structure definition

The end of the source file has been reached as defined with the END
directive, but a structure started with the STRUC directive has not been
ended with the ENDS directive. For example,

   STRUC x
   val1 DW   ?
        END        ;no ENDS before it 

This usually happens when you type END instead of ENDS to end a structure
definition.
