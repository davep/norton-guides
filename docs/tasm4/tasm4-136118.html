CODEPTR          Return procedure size                               Operator

  CODEPTR expression

    Returns the default procedure address size depending on the current
    memory model (WORD for models with NEAR code, DWORD for models with
    FAR code). CODEPTR can be used wherever DATAPTR is used.

    Example:    call    [codeptr bp + 8]
