@Interface              Return MODEL interface              Predefined equate

    The numeric equate @Interface provides information about the language
    and operating system selected by the MODEL statement.

    Values in
    bits 0-6    Meaning
        0       NOLANGUAGE (assembler)
        1       C
        2       SYSCALL
        3       STDCALL
        4       PASCAL
        5       FORTRAN
        6       BASIC
        7       PROLOG
        8       CPP (C++)

    Bit 7 can have a value of 0 (zero) for DOS/Windows, or 1 for Windows
    NT (1 for OS2 !?).

    For example, a value of 81h for @Interface shows that the Windows NT
    operating system and the C language were selected in the MODEL
    statement.
