Segment alignment not strict enough

The align boundary value supplied is invalid. Either it is not a power of
2, or it specifies an alignment stricter than that of the align type in
the SEGMENT directive. For example,

   SEGMENT data PARA
        ALIGN 32         ;error, PARA is only 16 
        ALIGN 3          ;error, not power of 2
