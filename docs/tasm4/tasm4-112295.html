UFARDATA         Define uninitialized far data segment              Directive

  UFARDATA [segname]                    ; Ideal or MASM mode
  .FARDATA? [segname]                   ; MASM mode

    Defines the beginning of an uninitialized far data segment in a
    module. If you don't specify a name, TASM uses segment name FAR_BSS.
    You can have more than one far uninitialized data segment per module.

    Example:    model huge
                irpc num,12345678
                     ufardata uf&num
                     db  0ffffh dup (?)
                endm
                ;...
