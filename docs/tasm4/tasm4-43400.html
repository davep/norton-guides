EVENDATA         Word-align data                                    Directive

  EVENDATA

    EVENDATA aligns evenly by advancing the location counter. If the
    location is odd when this directive appears, TASM emits an
    uninitialized byte, which is useful for uninitialized segments.

    Aligning data on word boundaries makes memory fetches more efficient
    on processors that use a 16-bit data bus.

    EVENDATA should not be used in byte-aligned segments.
