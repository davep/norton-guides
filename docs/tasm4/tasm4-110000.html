TESTFLAG         Optimized TEST                                     Extension

    Optimized form of the TEST instruction that tests bits with the
    shortest possible instruction.

    Example:    RECORD state  zone:4, chg:1, ins:1=1, bak:1=0, onoff:1
                ;...
                testflag edx,MASK ins           ; test dl,4
