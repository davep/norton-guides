        8255A-5 Programmable Peripharal interface

        The 8255A-5 features three independently accessible I/O
        ports that interface external devices to the bus circuitry.
        These external devices are the keyboard, speaker, configuration
        switcheds and cassette tape (if present). The 8255 supports
        several modes of peripheral in addition to the mode used by
        the PC.
