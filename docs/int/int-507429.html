        Disable mouse driver

        Entry:
          AX = 001fh

        Return:
          AX = 001fh successful
                ffffh unsuccessful
          EX:BX = vector for int 33h before mouse driver was first installed

        Note:   Restores vectors for Int 10h and Int 71h (8086) or
                Int 74h (286/386) if you restore Int 33h to ES:BX, driver
                will be completely disabled
