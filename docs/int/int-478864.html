        Communicating Applications Spec - Get event date

        Entry:
          AX = CB0Ah
          BX = event handle
          DL = queue
              00h task queue
              01h receive queue
              02h log queue
        Return:
          AX = 0000h successful
                  CX = year
                  DH = month
                  DL = day
             < 0    error code (see AX=CB01h)
