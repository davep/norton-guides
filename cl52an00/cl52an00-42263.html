 Internal Error # 5333 / VMIF 
 Misuse of VMM system
──────────────────────────────────────────────────────────────────────────────────

 Internal Error 5333 / VMIF occurs when there is a misuse of the virtual
 memory system by either an internal or external ( 3rd party product )
 subsystem.

 The following suggestions may eliminate the problem:

     Check to see if the user is assigning an incorrect data type to
     a IAMBS or GET instance variable.  This is a known cause for
     this error.

     If the user is using any third-party libraries, see if the error
     still occurs without the library.

     If the user is using a third-party linker, see if the problem
     occurs with .RTlink.
