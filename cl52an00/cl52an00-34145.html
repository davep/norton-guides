 Internal Error # 4424
 Error creating temp file
──────────────────────────────────────────────────────────────────────────────────

 IE 4424 indicates an error while creating a temporary file for indexing or
 sorting.  It can be caused by insufficient file handles, an invalid
 TEMPPATH setting, or insufficient network rights for the directory where
 temporary files are being created.  Temp files are created in the current
 directory if no TEMPPATH is specified.

 Action:  Correct the environment.  If insufficient network rights is
 the problem, either give the user the necessary rights, or redirect
 the temporary file to a more appropriate directory.
