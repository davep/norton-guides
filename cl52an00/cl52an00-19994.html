 Internal Error #  415
 Cannot open overlay file
──────────────────────────────────────────────────────────────────────────────────

 Internal Error 415 'Cannot open overlay file' occurs when a Clipper
 application cannot find or cannot open an overlay file.  Overlay files
 include .EXE files, .PLL files and .OVL files.

 The most common cause of this error is insufficient file handles available
 to the Clipper application.  Increase the File handles in the CONFIG.SYS
 and the SET CLIPPER=F:NNN.

 If insufficient files handles is not the problem, it is possible that the
 Clipper application cannot find the file it is trying to open.
