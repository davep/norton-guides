Color don't care       index 07h  GRAPH

Layout  b3-b0  CDC  Color don't care register

        Color don't care is used in conjunction with color compare
        mode (read mode #1). This register masks particular display
        memory planes from being tested during color compare cycles.

        0 : Do not consider the corresponding bit in the Color compare
            register and the data from the corresponding bit plane in
            the comparison.
        1 : Consider the corresponding bit in the Color compare
            register and the data from the corresponding bit plane in
            the comparison.
