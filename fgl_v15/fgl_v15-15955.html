  DEMO version ! After registering you will receive a complete version.

Description

This function returns information about the mouse state, that is :
the current screen position of the pointer, the state of the left and
right mouse buttons, the visibility status of the mouse pointer and
the version of the mouse driver.
If the version of the mouse driver is 0, that indicates that
the mouse is not connected or is configured incorrectly.


Syntax

   FGLMState() -> MouseState.
