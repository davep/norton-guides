General information

I have made this library to make it easy for CA-Clipper programmers to
incorporate neat graphics into their CA-Clipper applications. This library
offers the same, and sometimes more, than its commercial counterparts, like
Light Lib, Fast and CLGraph.

All the functions are easy to use.
With the redirected General Terminal API it is possible to make your
applications graphical without changing a single line of code.
Just add a call the function FGLSetMode() in the beginning of your code, and
all the output to your screen becomes graphical ! (Achoice(), MemoEdit,
Tbrowse, SAY, GET, etc, etc).

I have spend many, many hours making this library. Use it, and if
you like it : PLEASE REGISTER !

As a last remark I just hope that this library is just the library you
were looking for. Please distribute this shareware version as much as
possible !
