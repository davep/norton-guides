History.


Version 1.5 :

 - Well, first I rewrote my whole kernel so it supports only VESA. I also
   included much more modes.
   If your videocard doesn't have VESA, please run UNIVESA.
 - Included 1280x1024 graphic mode. The library now supports also 64k colors.
 - I put all graphic functions in one library. It was really stupid that
   you had to determine your color mode at link time in version 1.2 !
 - I removed the underscores in the names of my library functions. They just
   took up to must space of the 10 characters maximum available for function
   names.
 - I included BMP image support.
 - Also included mouse support.
 - Rewrote the total General Terminal API.
 - Included extended textmodes.
 - Sped up the 16 color mode routines. They now are FAST !
 - Included source for drawing windows and making wallpaper.
 - I really changed a lot of functions in this version. Some were removed
   and some were added. I advise you to totally read the Norton Guide so you
   can read the changes for yourself.


Version 1.2 :

 - Wrote Norton Guide to accompany software.
 - Library now supports 800x600 and 1024x768 graphic modes.
 - The file VGAFONTS.DBF is no longer needed. There are now four fonts
   available.
 - Paid much less attention to 16 color modes. Especially saving and
   restoring backgrounds is dead-slow. 16 color modes are maybe not supported ì◙ì*   in future releases.
