        DOS Internal - Get drive device list

        Entry:
          AX = 1501h
          ES:BX -> bufer to hold drive letter list (5 bytes per drive letter)
        Return:
          Buffer filled, for each drive letter
          BYTE  subunit number in driver
          DWORD address of device driver header
