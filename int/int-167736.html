        Port 041  -  8253-5 Channel 1

        Used to time the memory refresh cycle. Operates in mode 2
        (see port 043) with a divisor of 18, causing an interrupt
        to the 8237 DMA controller every 15.08 ms. Do not disturb
        (can result in PARITY CHECK).
