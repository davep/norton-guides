        CMOS RAM Data Register (PS/2)

        This port is used in conjunction with the address register at
        port 070 to read and write the RT/CMOS RAM bytes.


        Address   RT/CMOS RAM
        000 RT    Seconds
        001 RT    Seconds Alarm
        002 RT    Minutes
        003 RT    Minutes Alarm
        004 RT    Hours
        005 RT    Hours Alarm
        006 RT    Day of week
        007 RT    Date of month
        008 RT    Month
        009 RT    Year
        00A RT    Status Register A
        00B RT    Status Register B
        00C RT    Status Register C
        00D RT    Status Register D
        00E       Diagnostics Status Byte
        00F       Shutdown Status Byte
        010       Diskette Drive Type
        011       First Fixed Disk Drive Type Byte
        012       Seccond Fixed Disk Drive Type Byte
        013       Reserved
        014       Equipment Byte
        015-016   Low and High Base Memory Bytes
        017-018   Low and High Expansion Memory Bytes
        019-031   Reserved
        032-033   Configuration CRC Bytes
        034-036   Reserved
        037       Date Century byte
        038-03F   Reserved - Password?
