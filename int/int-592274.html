        DOS Country Information (DCI)

        Offset Size   Contens
        ───────────────────────────────────────────────────────────────────
        00-01  Word   Date format
                        0 = m d y (USA)
                        1 = d m y (Europe)
                        2 = y m d (Japan)
        02-06  Bytes  Currency symbol string (ASCIIZ)
        07-08  Bytes  Thousands separator (ASCIIZ)
        09-0A  Bytes  Decimal separator (ASCIIZ)
        0B-0C  Bytes  Date separator (ASCIIZ)
        0D-0E  Bytes  Time separator (ASCIIZ)
        0F     Byte   Currency format
                        0 = currency symbol, value
                        1 = value, currency symbol
                        2 = currency symbol, space, value
                        3 = value, space, currency symbol
                        4 = currency symbol is decimal separator
        10     Byte   Number of digits after decimal in currency
        11     Byte   Time format
                        0 = 12-hour clock
                       10 = 24-hour clock
        12-15  DWord  Address of case map routine (AL=character, far call,
                      only AL>=80h)
        16-17  Bytes  Data-list separator (ASCIIZ)
        18-21         Reserved for future use
