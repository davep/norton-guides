TYPE             Typecast (Ideal) or return size (MASM)              Operator

  TYPE name1 name2                      ; Ideal mode

    Applies the type of an existing variable or structure member to another
    variable or structure member.


  TYPE expression                       ; MASM mode

    Returns a number indicating the size or type of expression (you can't
    do this in Ideal mode because types can never be described numerically):

    Type          Description
    BYTE          1
    WORD          2
    DWORD         4
    PWORD         6
    QWORD         8
    TBYTE         10
    SHORT         0FFFFh
    NEAR          0FFFFh
    FAR           0FFFEh
    struc/union   Size of a structure or union instance
    table         Size of a table instance
    proctype      Returns 0FFFFh if the proctype describes a near procedure,
                  or 0FFFEh for a far procedure
