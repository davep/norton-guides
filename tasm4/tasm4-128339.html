  ;comment

    The assembler ignores any text on a line after a semicolon outside
    a quoted string.

    Example:   mov     [bx],al         ; Store the modified character


    Note:      In Turbo Assembler's Ideal mode, the COMMENT directive is
               not legal. Replace it with, for example:

               IF 0                    ; Zero will never evaluate to true
                  Comments
                    here...
               ENDIF
