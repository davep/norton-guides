Can't address with currently ASSUMEd segment registers

An expression contains a reference to a variable for which you have not
specified the segment register needed to reach it. For example,


   ASSUME  ds:DSEG,es:NOTHING
   mov     si,[MPTR]     ;no segment register to reach XSEG

   SEGMENT XSEG
   MPTR    DW   ?
   ENDS
