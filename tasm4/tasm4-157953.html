@Model                  Return current MODEL                Predefined equate

    The numeric equate @Model contains a representation of the memory
    model currently in effect (defined with the MODEL directive).

       0       TPASCAL
       1       TINY
       2       SMALL or FLAT
       3       COMPACT
       4       MEDIUM
       5       LARGE
       6       HUGE
       7       TCHUGE
