%DEPTH           Set size of depth field                            Directive

  %DEPTH width

    The %DEPTH directive indicates how many columns to reserve for the
    nesting depth field in program listings. This depth field specifies
    the nesting level for INCLUDE files and macro expansions. The
    default width for this field is 1 column.

    If you specify a width of 0, this field does not appear.
