%SUBTTL          Specify listing subtitle                           Directive

  %SUBTTL "text"

    Sets subtitle in listing file to text.

    You can use as many %SUBTTL directives as you wish. Each directive
    changes the subtitle that will appear at the top of the next listing
    page.
