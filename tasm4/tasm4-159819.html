??version               Get TASM version                    Predefined equate

    This predefined equate returns the TASM version. The high byte is the
    major version number, and the low byte the minor version number. For
    example, TASM version 3.2 would return a value of 320h.

    You can check the return value of this numeric equate to use TASM
    version-specific features.

    ??version reflects the true TASM version number, not the emulation
    version set by a VERSION statement.
