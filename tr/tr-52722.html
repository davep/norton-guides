 INMONTHS(<D start date>, <D end date>)
 Evaluates the number of months between two dates.
 Returns <expN> number of months between two dates.

 Returns -1 if start date is greater than end date.

 Useful in applications that handle dates by months.

 date1 = CTOD("08/07/86")
 date2 = CTOD("08/17/86")
 date3 = CTOD("11/26/86")

 INMONTHS( date1, date2 ) returns   .35
 INMONTHS( date1, date3 ) returns  3.67
 INMONTHS( date2, date3 ) returns  3.35
 INMONTHS( date3, date1 ) returns -1
