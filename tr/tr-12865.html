 BIN(<expN>)
 Converts a decimal number to binary (base 2).
 Returns <expC> binary value of <expN>.

 <expN> is converted to a positive number if negative.

 BIN(  16 ) returns "10000"
 BIN(  65 ) returns "1000001"
 BIN(   3 ) returns "11"
 BIN( 200 ) returns "11001000"
