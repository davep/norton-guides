 CHRFOUND(<C string1>, <C string2>)
 Evaluates whether <string2> contains all the characters in <string1>.
 Returns <expL> true if each character in <string1> occurs in <string2>.

 <string1> contains characters in any order for which to search.
 <string2> is the string in which to search for <string1> characters.

 Distinguishes between uppercase and lowercase alpha characters.

 CHRFOUND( "cAb", "AAABBaCdda" ) returns .F.
 CHRFOUND( "CaB", "AAABBaCdda" ) returns .T.
 CHRFOUND( "dddddd", "AAABBaCdda" ) returns .T.
 CHRFOUND( "b", "AAABBaCdda" ) returns .F.
