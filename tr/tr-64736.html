 LJUST(<expC>)
 Left justifies the contents of a string within that string.
 Returns <expC> with leading blanks moved to the end.

   LJUST("      Hello, world!")
 returns "Hello, world!      "

   LJUST("      Hello, world!     ")
 returns "Hello, world!           "
