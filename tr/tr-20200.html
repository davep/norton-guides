 CEILING(<expN>)
 Obtains an integer greater than <expN>.
 Returns an integer just above <expN>.

 Useful when both positive and negative numbers must round
 in the same direction.

 CEILING(  2.1 ) returns  3
 CEILING(   .3 ) returns  1
 CEILING( -0.3 ) returns  0
 CEILING( -2.9 ) returns -2

 * Evaluate number of pages in output
 page_len = 60   && lines per page
 no_pages = CEILING(LINES(memvar,65) / page_len)
