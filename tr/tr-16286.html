 BOQTR(<expD>)
 Evaluates the beginning of the current calendar quarter.
 Returns <expD> first day of calendar quarter in which the parameter falls.

 Return date    Dates between
 ═══════════    ══════════════
  01/01/yy      01/01 -- 03/31
  04/01/yy      04/01 -- 06/30
  07/01/yy      07/01 -- 09/30
  10/01/yy      10/01 -- 12/31

 * Compute beginning of current quarter.
 m_quartbeg = BOQTR( DATE() )
