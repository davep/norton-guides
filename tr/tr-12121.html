 BASE10(<N base>, <C base number>)
 Converts a different base value to decimal number (base 10).
 Returns <expN> decimal number (base 10) of <base number>.

 Maximum <base> is 36, in the format "0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ"

 BASE10(  2, "11111111" ) returns 255
 BASE10(  2, "1100100"  ) returns 100

 BASE10( 16, "FF" )       returns 255
 BASE10( 16, "64" )       returns 100

 BASE10( 32, "VV" ) returns 1023
 BASE10( 32, "7V" ) returns  255
 BASE10( 32, "34" ) returns  100

 BASE10( 36, "ZZ" ) returns 1295
 BASE10( 36, "FF" ) returns  555
 BASE10( 36, "64" ) returns  220

 Hazard  Does not handle negative numbers.
