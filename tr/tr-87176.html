 RJUST(<expC>)
 Right justifies the contents of a string within that string.
 Returns <expC> with trailing blanks moved to the beginning.

   RJUST("Hello, world!      ")
 returns "      Hello, world!"

   RJUST("      Hello, world!     ")
 returns "           Hello, world!"
