Int 2F Fn 1684  - Ms Windows - Pharlapx - Get Api Entry Point              [W]

   AX = 1684h
   BX = 28A0h (virtual device ID for PHARLAPX device) (see #2329)
   ES:DI = 0000h:0000h

Return: ES:DI -> VxD API entry point (see #2381)
         0000h:0000h if the VxD does not support an API
