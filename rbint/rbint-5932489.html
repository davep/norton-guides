Int 7F Fn 15  - Alloy Ntnx, Mw386 - Get And Clear Acknowledge Status       [N]

   AH = 15h
   CL = 04h
   DX:DI -> status structure (see #3631)

Return: AL = status
       00h successful
       DX:DI structure filled
       01h busy
       0Ah no acknowledgement has arrived

See Also: AH=12h,AH=14h/CL=01h

Format of Alloy status structure:
Offset Size    Description (Table 3631)
 00h   BYTE    sender ID
 01h   BYTE    channel number
 02h  4 BYTEs  receiver status (see #3625)
