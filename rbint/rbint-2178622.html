Int 21 Fn 2B  - Bitfossi - Installation Check                              [S]

   AH = 2Bh
   CX = 6269h ('bi')
   DX = 7449h ('tI')

Return: AL = FFh if not installed
   AL = 53h ('S') if bitFOSSI is installed
       CX = 444Eh ('DN')
       DX = 2D46h ('-F')
       ES = resident code segment
       ES:DI -> identification data
