Int 69 Fn 0103  - Decnet Dos Cterm - Status                                [N]

   AX = 0103h
   DX = session handle

Return: AH status flags (see #3381)
   AL = reason code if DECnet error (see #3382)

See Also: AX=0104h

Bitfields for DECnet DOS CTERM status flags:
Bit(s) Description (Table 3381)
 7 session has been aborted
 6 DECnet error
 1 trace data available
 0 receive data available
