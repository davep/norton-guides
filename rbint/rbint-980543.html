Int 14 Fn 4005  - I1541 - Send Secondary Address For Listen                [t]

   AX = 4005h
   BL = channel number and mode (see #0281)

Return: CF clear if successful
   CF set on error
       AL = error number (see #0280)

See Also: AX=4004h,AX=4006h
