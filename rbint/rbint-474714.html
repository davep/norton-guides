Int 10 Fn 007F  - Video - Paradise Vga, At&t Vdc600 - Enter Color Vga Mod  [V]

   AX = 007Fh
   BH = 07h

Return: AL = 7Fh if successful (AT&T VDC600)
   BH = status (Paradise/Dell)
       00h invalid function
       7Fh successful

Note:  this function also switches to video mode 3

See Also: AX=007Eh,AX=007Fh/BH=03h,AX=007Fh/BH=06h,AH=FFh"Oak"
