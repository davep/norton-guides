Int 1A Fn 0B  - Time - Set System-timer Day Counter (xt2,ps)               [B]

   AH = 0Bh
   CX = count of days since Jan 1,1980

Return: CF set on error
   CF clear if successful

See Also: AH=05h,AH=0Ah
