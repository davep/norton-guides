Int 7A Fn 0200  R - Autocad Device Interface - Rendering Device - Initial  [A]

   AX = 0200h
   BX = 0000h
   CX = interface level (0001h)

Return: BX = 0000h (driver uses packet communication)
   DX:CX -> packet-mode entry point (see #3503)
