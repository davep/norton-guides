Int 67 Fn DE06  - Virtual Control Program Interface - Get Phys Addr Of Pa  [E]

   AX = DE06h
   CX = page number (linear address shifted right 12 bits)

Return: AH = status
       00h successful
       EDX = physical address of page
       nonzero invalid page number (AH = 8Bh recommended)

See Also: AX=5BE0h
