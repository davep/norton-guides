Int 1A Fn 05  - Sperry Pc - Set Real-time Clock Date                       [b]

   AH = 05h
   CL = year-1980
   CH = 00h (???)
   DH = month (binary) (01h-0Ch)
   DL = day (binary) (01h-1Fh)

Return: nothing
