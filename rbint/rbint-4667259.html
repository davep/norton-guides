Int 31 Fn FF2D  P - Causeway V1.3 - "resmemso" - Resize Selector:offset M  [E]

   AX = FF2Dh
   SI:DI = selector:offset for memory block
   CX:DX = new size of block

Return: CF clear if successful
      SI:DI = new selector:offset for memory block
   CF set on error

See Also: AX=FF00h,AX=FF2Ch,AX=FF2Eh
