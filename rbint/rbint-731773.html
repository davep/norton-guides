Int 10 Fn D001  - S3vbe/core2.0 V3.00+ - Get Active Options                [V]

   AX = D001h

Return: AX = FFFFh if supported
       BX = currently active options (see #0154)

See Also: AX=D000h"S3VBE",AX=D002h"S3VBE"

Bit(s) Description (Table 0154)
