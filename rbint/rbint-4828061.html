Int 4D Fn 00  - Ti Professional Pc - Disk - Reset Disk System              [B]

   AH = 00h
   DL = drive (if bit 7 is set both hard disks and floppy disks reset)

Return: AH = status (see #0166 at INT 13/AH=01h)
   CF clear if successful (returned AH=00h)
   CF set on error

Note:  this function is the same as INT 13/AH=00h on a standard PC BIOS

See Also: AH=01h,AH=02h,AH=08h,AH=0Bh,INT 13/AH=00h,INT 46"TI Professional"
