Int 6F Fn 0408  - Hp Vectra Ex-bios - "f_io_ctrl Sf_crv_wr_prompts" - Wri  [b]

   AX = 0408h
   BP = 0114h (driver ID for V_HPHIL)
   BX = device address indicator (see #3431)
   DH = HP HIL device major address
   DL = prompt number

Return: AH = status (see #3415)
   BP,DS destroyed

See Also: AH=00h"HP Vectra",AX=020Eh,AX=0406h/BP=0114h,AX=040Ah/BP=0114h
