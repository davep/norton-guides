Int 10 Fn 4F04  - Vesa Supervga Bios - Save/restore Supervga Video State   [V]

   AX = 4F04h
   DL = subfunction
       00h get state buffer size

       Return: BX = number of 64-byte blocks needed
       01h save video states
       ES:BX -> buffer
       02h restore video states
       ES:BX -> buffer
   CX = states to save/restore (see #0083)

Return: AL = 4Fh if function supported
   AH = status
       00h successful
       01h failed

Bitfields for VESA SuperVGA states to save/restore:
Bit(s) Description (Table 0083)
 0 video hardware state
 1 video BIOS data state
 2 video DAC state
 3 SuperVGA state
