Int 15 Fn DE0C  - Desqview V2.00+ - "getmem" - Allocate "system" Memory    [Q]

   AX = DE0Ch
   BX = number of bytes

Return: ES:DI -> allocated block or 0000h:0000h (DV 2.26+)

Note:  use SETERROR (AX=DE15h) to avoid a user prompt if there is insufficient
     system memory

See Also: AX=1001h,AX=102Eh,AX=DE0Dh,AX=DE15h,AX=DE19h
