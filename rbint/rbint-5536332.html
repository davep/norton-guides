Int 67 Fn DE00  - Virtual Control Program Interface - Installation Check   [E]

   AX = DE00h

Return: AH = status
       00h VCPI is present
       BH = major version number
       BL = minor version number
       nonzero  VCPI not present
BUG:   MS Windows 3.00 is reported to "object violently" to this call.

See Also: INT 2F/AX=1687h
