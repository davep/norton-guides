Int 21 Fn C703  - Novell Netware - Transaction Tracking System - Abort Tr  [N]

   AX = C703h

Return: CF clear if successful
       AL = 00h
   CF set on error
       AL = error code (FDh-FFh) (see #1497)

Note:  this function is supported by NetWare 4.0+ and Advanced NetWare 1.0+
