Int 10 Fn 6A02  - Dgis - Inquire Int 10 Output Device                      [V]

   AX = 6A02h
   ES:DI = 0000h:0000h

Return: ES:DI = 0000h:0000h  if current display is non-DGIS
       else address of the current DGIS INT 10 display
