Int 10 Fn 02  - Video - Set Cursor Position                                [V]

   AH = 02h
   BH = page number
       0-3 in modes 2&3
       0-7 in modes 0&1
       0 in graphics modes
   DH = row (00h is top)
   DL = column (00h is left)

Return: nothing

See Also: AH=03h,AH=05h,INT 60/DI=030Bh,MEM 0040h:0050h
