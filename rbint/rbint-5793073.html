Int 7A Fn 09  - Ibm 3270 Workstation Program Api - Window Services         [I]

   AH = 09h
   BX = 8020h (synchronous request)
   CX = 00FFh
   DX = ID of window service controller (WSCTRL)
   AL = service number (see #3507)
