Int 6A Fn D0  - Decnet Dos Local Area Transport - Open Session             [N]

   AH = D0h
   DH = FFh
   AL = password flag
       FFh no password
       0Fh password at ES:DI
   ES:BX -> LAT session control block (see #3398)
   ES:DI -> 16-byte blank-padded password (optional)

Return: AH = 00h success
       DL = handle

See Also: AX=D000h

Format of LAT Session Control Block:
