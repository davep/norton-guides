Int 6F Fn 03  - 10net - Get Address Of Configuration Table                 [N]

   AH = 03h
   DS:DI -> node ID (optional)

Return: ES:BX -> configuration table (see #3427)

See Also: AH=13h,INT 21/AX=5E01h"10NET"
