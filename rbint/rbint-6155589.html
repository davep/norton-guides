Int E0  - Real/32 - "c_assign" - Assign Default Console To Another Proces  [O]

   CL = 95h
   DS:DX -> Assign Control Parameter Block (ACPB) (see #3778)

Return: AX = status

See Also: INT E0/CL=94h,INT E0/CL=99h

Format of REAL/32 Assign Control Parameter Block (ACPB):
Offset Size    Description (Table 3778)
 00h   BYTE    ID of console to assign
 01h   BYTE    flag: if FFh, new process must have CNS as console for this
         call to succeed
 02h   DWORD   -> process descriptor (see #3768) or 00000000h
 06h  8 BYTEs  name of process to search for if descriptor field above is zero
