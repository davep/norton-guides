Int E0  - Real/32 - "m_free" - Deallocate Memory                           [O]

   CL = 81h
   DS:DX -> Memory Free Parameter Block (MFPB) (see #3776)

Return: AX = status (0000h success, else error code)

See Also: INT E0/CL=81h

Format of REAL/32 Memory Free Parameter Block (MFPB) :
Offset Size    Description (Table 3776)
 00h   WORD    starting segment of block to be freed
 02h   WORD    reserved (0)
