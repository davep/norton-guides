Int 1A Fn 97  - Pcmcia Socket Services V1.00 - Set Error Detection Code    [X]

   AH = 97h
   AL = adapter number
   BH = EDC generator
   BL = socket number
   DH = EDC attributes (see #0614)
   DL = EDC type (see #0613) (only one bit may be set)

Return: CF clear if successful
       AH destroyed
   CF set on error
       AH = error code (01h,02h,04h,0Bh) (see #0580)

See Also: AH=96h,AH=9Ch
