Int 10 Fn A00D  - Ati Mach64 Bios - Get Sleep Mode                         [-]

   AX = A00Dh

Return: AH = 00h (successful)
   CL = current power mode
       00h normal operation
       01h standby
       02h suspend
       03h off

See Also: AX=A00Ch,AX=4F10h/BL=02h
