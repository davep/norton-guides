Int 67 Fn 5D05  - Nanosoft Md386 - Get Internal Handle Table               [m]

   AX = 5D05h
   BX = handle number
   ES:DI -> 1024-byte buffer for handle table

Return: AH = status (see #3319)
   buffer filled if AH=00h

Note:  used for debugging purposes
