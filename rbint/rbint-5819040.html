Int 7C Fn 0001  - Sk-upps Data Link Interface Api - Down Driver            [N]

   AX = 0001h

Return: AX = completion code (see #3517)

Note:  the DLI will refuse to unload if there are any protocols
     (see AX=0007h) or clients (see AX=0010h) active.
