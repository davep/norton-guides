Int 16 Fn E006  - Ami Bios - Bios-flash Interface - Flash Write Protect    [b]

   AX = E006h

Return: CF clear if successful
       AL = FAh
   CF set on error

Note:  this function performs any delay required to allow the Flash ROM to
     stabilize in the write-protected state

See Also: AX=E000h,AX=E004h,AX=E007h
