Int 2F Fn F202  - Winx - Set Server's Working Directory                    [W]

   AX = F202h
   BX:CX -> directory

Return: AX = status
       FFFFh if WINX is busy processing a different request
       0000h if successful

See Also: AX=F200h,AX=F203h
