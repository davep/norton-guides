Function 32h            Get DPB                                      DOS 5.0+

    Retrieves drive parameters for the specified drive.

        Entry   AH = 32h
                DL = Drive number (0 = default, 1 = A, etc.)

        Return  AL = 00h   DS:BX = address of DPB structure.
                AL = FFh   Error, invalid drive or disk error
