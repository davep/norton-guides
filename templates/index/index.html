{# -*- engine: jinja2 -*- #} davep's Public Norton Guide Collection

Because of the number of Norton Guide tools I've published, on occasion I get asked if I have a copy of a particular Norton Guide file. I've never been that comfortable with just handing out guides that are still licenced in a way that doesn't allow distribution. Meanwhile, for the longest time, were a couple of very large collections of guides out there on the web that people could go to, but it looks like they're starting to disappear; at least one that was built using ng2html seems to have gone offline.

With this in mind, I'm going to populate this little site with all the guides I have that I have a licence to publish in public, or which appear to have a licence that allowed in in the spirit of shareware. Let me repeat and make clear that part: The only guides that are published here are guides that permit distribution and republishing in some way.

If there is a particular guide you are after, and it isn't listed here, there's little point in contacting me and asking for a copy; as much as I'd love to help I would prefer not to distribute files I don't have a licence to distribute.

If you want to know what guides I do have to hand, and especially if you can help gain permission to publish any of them, please see the Wiki for this project and perhaps start a discussion.
