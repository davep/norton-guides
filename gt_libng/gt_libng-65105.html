 Syntax

        GT_PrintFile( nDown , cWhat , nLeftMgn , nHandle )

 Arguments:

      nDown    - Number of rows to move down
      cWhat    - Text to be output
      nLeftMgn - Left margin of text
      nHandle  - Handle of text file

 Returns:

      NIl
