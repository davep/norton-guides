 Mark a record as deleted
──────────────────────────────────────────────────────────────────────────────

 Syntax

        GT_DelRec(<lMove>) => lSuccess

 Arguments:

      <lMove> should the system move to another record
      after the deletion ?

 Returns:

      .T. / .F.

 Description:

      Mark a record as deleted

 Examples:

      IF .NOT. GT_DelRec(.T.)
          ? 'Error'
      ENDIF
