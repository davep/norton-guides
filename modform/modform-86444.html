 CMF File Structure

 The Soundblaster CMF (Creative Music File) stores instrument and music
 data for the Soundblaster FM organ. Two related file formats are the SBI
 (Sound Blaster Instrument) and IBK (Instrument Bank) which both store only
 instrument information.

 The CMF file consistes of three blocks - a header block which contains
 version and related information, an instument block which contains
 instrument parameters for the FM chips, and the data section which
 contains music data.
