 Variable Window
 Greatly Enhanced
──────────────────────────────────────────────────────────────────────────────

 Shows you the variables currently available. Each different variable type
 local, static, privates and publics may be toggled on or off with a simple
 keystroke. Alt-L for locals, Alt-S for statics, Alt-P for publics and Alt-V
 for privates.

 The first level of arrays are shown, character strings can be shown in
 Hexadecimal, string lengths are shown, variables passed by reference can be
 viewed/edited.  Local variable that have not been used may be shown and
 variables that have been changed during the last step can be highlighted.
