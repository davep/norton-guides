 Inspecting Simply Variables
 Strings, Dates, Numerics and Logicals
──────────────────────────────────────────────────────────────────────────────

 To edit the contents of a variable you first need to change to the variable
 window. The quickest way of doing this is by pressing <ALT><3> and using a
 combination of the ↑ or ↓ keys to move the highlight bar to the variable
 that you wish to edit and press ◄──┛.

 You will see a box similar to the following appear:


   ┌─ VAR: CFILENAME ────────────────────────────────────────────────┐
   │"INVOICE"                                                        │
   └───────────────── Ascii: 34  Pos: 0 ──── LEN: 7 ─────── TYPE: C ─┛
                      └───┬───┛  └──┬─┛      └──┬─┛         └──┬──┛
          ASCII value of ─┛         │           │              │
          character under      Position of    Length       Variable
          cursor               cursor in      of item          Type
                               edit box


 This will allow you to edit the variable. Changes maybe made by typing over
 the variable and confirmed by pressing ◄──┛. The <ESC> key will cancel any
 changes.
