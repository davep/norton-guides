 Window Menus
 Using the mouse with the window menus.
──────────────────────────────────────────────────────────────────────────────

 Each window has it's own menu associated with the window.

 You may bring up a pop-up menu on a window, by moving the mouse cursor to
 the window that you wish to perform an action on, by holding down the right
 mouse button.
