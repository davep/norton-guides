 Workarea Status
 Display information about the currently selected workarea.
──────────────────────────────────────────────────────────────────────────────

 This option will add a new window to MrDebug. This window will contain basic
 information about the currently selected work area. This information
 includes such items as the alias name, the work area number, the current
 record number, the current index name, the current index key value and the
 index key.

 You can move this window and re-size it to suit your needs. If you change
 the height of the window, you will reveal the contents of the current
 record.

 The hot-key combination for this item is <Alt><F6>.
