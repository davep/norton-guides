 The PPO Window                                                        <Alt-7>
 About the PPO window.
──────────────────────────────────────────────────────────────────────────────

 The pre-processed code window is used to display the output of code compiled
 with the /p switch.

 The PPO window will appear next to the source code window. You may move or
 re-size this window to suit your needs.

 Below is an example of the PPO window:



  ┌─ 7 PPO (RCV\RCV.PPO) ──────────────────────── (76, 1) ─┐
  │   69:RETURN (NIL)                                      ↑
  │   70:                                                  ▒
  │   71:                                                  █
  │   72:STATIC FUNCTION GetSet_Settings()                 ▒
  │   73:                                                  ▒
  │   74:   LOCAL aSettings := SaveCurtain()               ▒
  │   75:                                                  ▒
  │   76:   SET(5, 1960)                                   ▒
  │   77:   SET(4, "dd/mm/yy")                             ▒
  │   78:   SET(16, 0)                                     ▒
  │   79:   SET(32, .F.)                                   ↓
  └←██▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒→┛


 As your program moves through the source code, the PPO window will change to
 reflect the pre-processed output for the source code.

 If you move or re-size the PPO window don't forget to save the changes,
 otherwise the window will return to it's original place the next time you
 run MrDebug.

 The following is an example of the pop-up menu for the PPO window, which
 may be summoned either by clicking the right mouse button on the window
 or by using <Alt-F10>.
