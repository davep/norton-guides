 About Style
 A brief description about styles
──────────────────────────────────────────────────────────────────────────────

 MrDebug allows you to define up to five styles of which you may select a
 style to use or change at any point.

 By default, the first time that you use MrDebug Style number 1 will be the
 default style. When you move, re-size the windows and save the settings you
 save the settings of the window positions and sizes to style number 1.

 To use another style you may simply select another style form the Styles
 menu. Any changes that you may to the window sizes or position when saved
 willbe stored under the currently selected style.

 If a windows does not have any predefined co-ordinates then when the window
 opens it will appear in Zoom mode, taking up as much screen as possible,
 forcing you to give it some pre-defined co-ordinates which you should then
 save.
