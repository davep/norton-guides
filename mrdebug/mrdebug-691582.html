 Disable Break Points
 Ignore all Break Points
──────────────────────────────────────────────────────────────────────────────

 This option allows you to run your program without any interference from
 break points that you have set within the program.

 This option saves you from deleting all of your break points. Toggling this
 option, to on, will allow you to keep your break point within the program,
 but have the program simply ignore the command to stop execution.

 You may disable all break points from the Command window by using the
 DISABLE BP command. You may disable a single break point by pressing
 <Alt><Enter> on the line that contains the break point - this will toggle a
 break point from being enabled to Disabled, pressing <Alt><Enter> again will
 re-enable the break point.

 If the current line shares a break point with a trace point and/or a pass
 point you may disable a single item by listing that particluar item, using
 the LIST <BP>|<TP>|<WP>|<PP>|<ALTD> command to list the items, and then
 cursor down to the line that contains the curren item and by pressing
 <Alt><Enter>.
