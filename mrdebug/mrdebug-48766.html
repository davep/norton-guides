 Shift-Alt-F - Function
 Search for function/procedure/method.
──────────────────────────────────────────────────────────────────────────────

 This option searches specifically for a function, procedure, method, or
 class declaration

 This option will display a box where you may enter the name of a function,
 procedure, method or class declaration to search for in the current source
 code file. It will not find partial names.

 You need not supply the fullname of the function, procedure etc, simply the
 first few letters of the name and MrDebug will find the first occurrence of
 the text. If the text that is found is no the text that you were looking for
 then you can use the Again keys to search for the next occurrence.

 Any thing that you enter in this box will be made available to the Text
 Search option.

 This is affected by the Case Sensitivity option.
