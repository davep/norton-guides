 Assert/Trace Window
 Toggles the displaying of the Assert/Trace window
──────────────────────────────────────────────────────────────────────────────

 This option will toggle the display of the Assert/Trace window.

 With this option toggled to on, the Assert/Trace window is shown, with this
 option not toggled, the window will not be displayed.

 By default the Assert/Trace window is not shown.

 The Assert/Trace window will be displayed automatically when an Assertion or
 Trace is triggered, and you have the Auto Open Trace Window toggled to on
 under the Assert Menu.
