 About MrDCLWin.lib
 The CLWindow object module
──────────────────────────────────────────────────────────────────────────────

 MrDCLWin.lib is the CLWindow compatibility module that should be linked in
 when using MrDebug with CLWindow.

 MrDCLWin.lib provides some extra support for when you are using MrDebug with
 CLWindow. It saves and restores the current window as well as preserving the
 status of the current window.

 It also replaces SC_MAXROW() and SC_MAXCOL() so that these functions reflect
 the application window size, not the screen size.
