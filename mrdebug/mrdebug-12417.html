 F2 - Open Source
 Open a new file.
──────────────────────────────────────────────────────────────────────────────

 This option will display a pop-up window where you may enter the path and
 name for a source code file. The default name for the source code file is
 the current .PRG. You may enter a drive letter and path before the name of
 the source code file.

 The file need not be purely CA-Clipper related .PRG code. This option will
 allow you to open any ASCII file.
