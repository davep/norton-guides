 Style
 Using Styles.
──────────────────────────────────────────────────────────────────────────────

 This setting controls a toggle for the method of auto-arranging windows.
 MrDebug supports two methods of auto arranging:

    1: the source code window takes up the whole width of the screen
    2: the watch, variable call stack take over the right hand side of the
       screen

 This option only works whilst you haven't moved any windows. When you start
 moving windows, this option will only work if you use the Reset All option
 first.

 The hot-key combination for this item is <Alt><Y>.
