 List Watch Points
──────────────────────────────────────────────────────────────────────────────

 This option will display a list of all the watch points that you have set.

 The list will contain the name of the source code file and the line number
 that the watch point has been set on for each break point that you have set.

    █▀▀▀▀▀ Watch Point List ▀▀▀▀▀▀█     The ↑ or ↓ keys will allow you to
    █ LANTI                       █     choose a watch point, press ◄──┛ on
    █ MrdIsSymbol("Sx_AutoAhare") █     the watch point and the source code
    █ MrdIsSymbol("Sx_AutoOpen")  █     window will change to enable you to
    █▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄█     view the watch point.
                                    

 To restore the source code window, press <Shift><Alt><R>.

 This option will remain dimmed if there are no watch points set and you will
 not be able to select it.

 The Command window short-cut command for this is LIST WP.
