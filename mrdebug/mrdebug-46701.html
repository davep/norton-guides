 Shift-Alt-A - Again
 Repeat the last search.
──────────────────────────────────────────────────────────────────────────────

 This option allows you to search again. This option will only be available
 if you have searched for something previously. If you have not already
 performed a search this option will do nothing - apart from light a
 cigarette and have a cup of coffee.

 If a search is repeated and something is found, the source code window will
 move the to source code that it found.
