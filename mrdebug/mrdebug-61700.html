 Ctrl * - Find Next Construct point 
 Show the next part of a construct.
──────────────────────────────────────────────────────────────────────────────

 This option allows you to find the next part of a construct within the
 source code window.

 For example pressing <CTRL><*> on a FOR ... TO ... line will reposition the
 cursor on the NEXT line that matches the FOR construct
