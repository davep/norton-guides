 Shift-Alt-V - Toggle the display of the Profiler window
 Display or Hide the Profiler Window
──────────────────────────────────────────────────────────────────────────────

 This option toggles the setting for displaying or not displaying the
 Profiler window.

 With this toggle set to on, the profiler window is displayed. You may move
 this window and re-size it to suit your needs. If you save your setup or have
 auto save set to on, then the position and size of the window will be saved.

 With this option toggled to off, the Profiler window will not be shown.
 However, any information stored within the profiler window is not lost.
