 or to be re-sized. This section covers the basics of how MrDebug's windows

 Moving and re-sizing a window with the keyboard.

 To move or re-size a window change to the window that you want to move to,
 and press <Alt><M>. This selects that window for moving or re-sizing.

 At end point you may abort the size or move by pressing the <ESC> key. To
 confirm your actions you may press ◄──┛

 Once you have pressed <Alt><M>, the border of the window will change to show
 you which window you will be working on.

 You may use the following keys to move the window:

        ↑          - moves the window up
        ↓          - moves the window down
        ←          - moves the window left
        →          - moves the window right
        <Home>     - moves the window to the left hand side of the screen
        <End>      - moves the window to the right hand side of the screen
        <PageUp>   - moves the window to the top of the screen
        <PageDown> - moves the window to the bottom of the screen

 If you are happy with the position of the window then press ◄──┛, if you
 don't like the new position of the window, then hit the <Esc> key, but not
 with a hammer, and the window will be restored at it's original position.

 If you want to re-size a window the above key strokes apply, but add a
 <Shift> before them ie:

        <Shift> ↑        - shrinks the window, vertically
        <Shift> ↓        - expands the window, vertically
        <Shift> ←        - shrinks the window, horizontally
        <Shift> →        - expands the window, horizontally
        <Shift><Home>    - shrinks the window, horizontally - to a small size
        <Shift><End>     - expands the window, horizontally - to as large a
                           size as the right hand side of the screen allows
        <Shift><PageUp>  - shrinks the window, vertically, - to a small size
        <Shift><PageDown>- expands the window, vertically to as large as size
                           as is allowed by the size of the screen

 Moving a window with the Mouse

 You may use the mouse to move a window by moving the mouse pointer to top of
 the window or the left edge of the window, clicking, and holding down, the
 left mouse button.  This will change the border of the window. You may now
 drag the mouse to re-position the window. When you have moved the window to
 the position that you want, simply release the mouse button.

 Re-sizing a window with the Mouse

 To re-size a window with the use of a mouse, drag the mouse until the mouse
 pointer covers the bottom right hand corner of a window. Click and hold down
 the left mouse button and drag the mouse until the window is the size that
 you want it to be. When the window is the size you wish it to be, simply
 release the mouse button.

 Note: To make any changes made to windows permanent, you must save the
      changes. For further information about saving your settings, refer to
      the see also guide on this page.

 Each Window also has it's own pop-up menu which can be activated with
 <Alt><F10>, or the right mouse button. These menus contain items that are
 relevant to each separate window.
