 Create Profile Log
 Write the current Profiler details to a log file
──────────────────────────────────────────────────────────────────────────────

 This option will write the information to a text file and dump the contents
 of the profiler to the profiler log file. This is just a plain text ASCII
 file that contains the contents of the Profiler window.
