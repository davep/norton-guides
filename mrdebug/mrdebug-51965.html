 Shift-Alt-M - Memory/Info
 This displays the current memory settings.
──────────────────────────────────────────────────────────────────────────────

 This option will bring up a list of various memory values. This includes the
 standard settings such as MEMORY(0), MEMORY(1), MEMORY(2), MEMORY(3),
 MEMORY(4) and a few other weird and, of course undocumented values.

 If you, like the best of us, use Blinker 3, and have the function
 BLIMGRSTS() linked into your program you will be able to see extra
 information about the specific mode that you are running in.
