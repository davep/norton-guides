 The Watch/Trace/Track Window                                          <Alt-2>
 Using the Watch/Trace/Track window.
──────────────────────────────────────────────────────────────────────────────

 Shows you any watch points, trace points or track points that you may have
 set.

 This window will show you any variables or expressions that you have marked
 to be watched or traced either as Trace points, watch points, watch track
 points or trace track points.

 ┌─ A tp symbol represents a trace point
 │
 │    ┌─ 2 Watch/Trace/Track ───────────────────────────────────┐
 └────│lDecode <tp, L> : .F. // Was:Undefined                   ↑ 
 ┌────│asettings[34] <wp, C> : " ☼ ☼ ☼ ☼ ☼ ☼ ☼ ☼ ☼ ☼ ☼ ☼ ☼ ☼ ☼ ☼█ 
 │    │                                                         ↓ 
 │    └←██████████████████▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒→┛ 
 │                                                                
 └─ A wp symbol represents a watch point

 You may use the cursor keys to move up and down this window.

 <Delete>   key will remove items from this list.
 <SpaceBar> edits the currently highlighted item.
 <Insert>   creates a new item using the currently selected item in the
            window as a default, which you may edit.

 The following is an example of the pop-up menu for the Watch/Trace/Track
 window, which may be summoned either by clicking the right mouse button
 on the window or by using <Alt-F10>

     ├──────────────────────────────┤
     │Toggle Trace Point  Alt-Return│
     │Delete Watch/Trace         Del│
     │Delete All                    │
     ├──────────────────────────────┤
