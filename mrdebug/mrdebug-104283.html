 The Workarea Status Window                                            <Alt-6>
 About The Workarea Status Window.
──────────────────────────────────────────────────────────────────────────────

 The Workarea status window shows you some brief information about the
 current workarea, such as the workarea number, the alias name, the currently
 selected index, if any, the current record number, the index key and the
 actual index expression.

  ┌─ 6 Workarea 1 Status ───────────────┐ You may view further areas by using
  │  WorkArea:1 - MICHELLE              │ the <Page Up> and <Page Down> keys
  │    Record:333/1795                  │ to jump to the previous area and to
  │     Index:1 - E:\RCV\MICHELLE.NTX   │ jump to the next area. The <Home>
  │ Index Key:FILM+YEAR+RATING          │ key will take you to the first work-
  │Actual Key:"ZZZ9991983Hmmmmm....Yes" │ area. The <END> key takes you to
  └─────────────────────────────────────┛ the last used workarea.

 For these keys to work. The Workarea status window, must be the currently
 selected window.

 By default, the workarea window will change to reflect the current workarea,
 i.e. each time that the workarea changes, the workarea window will change to
 reflect the new work area.

 You may override this, by using any of the keys that affect the workarea
 window. The moment that you press a key, the window will not update to
 reflect any changes to workareas. It will only update to reflect changes to
 the workarea shown in the window. Should you re-size the window, then you
 will be able to see the contents of the record.

  ┌─ 6 Workarea 1 Status ───────────────┐
  │  WorkArea:1 - MICHELLE              │
  │    Record:333/1795                  │
  │     Index:1 - E:\RCV\MICHELLE.NTX   │
  │ Index Key:FILM+YEAR+RATING          │
  │Actual Key:"ZZZ9991983Hmmmmm....Yes" │
  │─────────────────────────────────────│
  │[ 1] DATEINPUT  : 23/12/93          ↑│ The Workarea status window may not
  │[ 2] FILM       : ZZZ999            █│ hold enough space for the entire
  │[ 3] YEAR       : 1983              ▒│ record, so scroll bars will appear
  │[ 4] REFERENCE  : 987123645         ▒│ that will allow you to scroll up
  │[ 5] STARS      : 00000001          ▒│ and down the fields using the ↑
  │[ 6] PARTNUMBER : 010               ▒│ and ↓ keys.
  │[ 7] RATING     : Hmmmmm....Yes     ↓│
  └─────────────────────────────────────┛


 To allow the window to continue updating as the workareas change just press
 <SPACEBAR>.

 The following is an example of the pop-up menu for the Workarea Status
 window, which may be summoned either by clicking the right mouse button
 on the window or by using <Alt-F10>.

     │Default Area  Space│
     ├───────────────────┤
     │Next Area      PgDn│
     │Previous Area  PgUp│
     │First Area     Home│
     │Last Area       End│
     ├───────────────────┤
     │Zoom          Alt-Z│
     │Close        Alt-F6│
     └───────────────────┛
