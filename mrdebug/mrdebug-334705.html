 Setting a Watch or Trace Point
 Allows you to watch or trace a variable or expression
──────────────────────────────────────────────────────────────────────────────

 A few quick notes:

   Trace point : will invoke the debugger each time that the trace point
                 changes and show you the old and new states of the
                 expression or variable.

   Watch point : will simply show the new value each time that the debugger
                 is invoked if the variable or expression changed.

 To set a watch point or a trace point on a variable, you may use the cursor
 keys to move the cursor in the source code window to the variable name and
 simply press <Alt><T> to set a trace point or <Alt><W> to set a watch point
 on that variable.

 For more complex expressions, you may highlight the expression using the
 Shift and ← or → keys and then press <Alt><T> to set a trace point or
 <Alt><W> to set a watch point

 If you are using a mouse, you may set a watch point by moving the mouse
 cursor to the start of the variable or expression and holding down the left
 mouse button whilst you drag the mouse cursor over the text highlighting the
 expression that will be set as a watch point when you let go off the left
 mouse button.
