 List Break Points
──────────────────────────────────────────────────────────────────────────────

 This option will display a list of all the break points that you have set.

 The list will contain the name of the source code file and the line number
 that the break point has been set on for each break point that you have set.

    █ TEST.PRG             166    █    choose a break point, press ◄──┛ on
    █ TEST.PRG             184    █    the break point and the source
    █ TEST.PRG             420    █    code window will change to enable
    █ TEST.PRG             464    █    you to view the break point.
    █ TEST.PRG             507    █ 
    █ TEST.PRG             506    █    To restore the source code window,
    █ _MRDDBFL.PRG         251    █    press <Shift><Alt><R>.
    █ _MRDDBFL.PRG         462    █ 
    █ _MRDDBFL.PRG         314    █ 
    █ _MRDERSY.PRG         172    █ 
    █ TEST.PRG             186    █ 
    █▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄█ 
                                    

 This option will remain dimmed if there are no break points set and you will
 not be able to select it.

 The Command window short-cut command for this is LIST BP.
