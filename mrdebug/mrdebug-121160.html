 About The Printer Window.
──────────────────────────────────────────────────────────────────────────────

 The Printer Output window shows you the results of any output that is
 directed at the printer or to a file.

 A popup menu on this window allows you to tailor some of the options
 that are available to suit your needs. For example, you may not want
 output that is being re-directed to a file to appear in this window. To
 stop this output from appearing your could enable the option 'Only for
 LPTx:/COMx:'.

 The popup menu also includes special break points for breaking before a
 page eject, a break on Print and a break on Margin overrun.

     ┌───────────────────────────────┐
     │    Eject                      │
     │[x] Only for LPTx:/COMx:       │
     ├───────────────────────────────┤
     │[x] Ruler                      │
     │    Space Character '·' (250)  │
     │    Eject Sequence '♀'         │
     │    Right Margin (0)           │
     ├───────────────────────────────┤
     │[ ] Break on Eject             │
     │[ ] Break on Print             │
     │[ ] Break on Margin Overrun    │
     ├───────────────────────────────┤
     │    Zoom                  Alt-Z│
     │    Close           Shift-Alt-=│
     └───────────────────────────────┛

 Note: To summon a pop-up menu, first select the window that you want
       and then use the right mouse bbutton to pop-up the menu. If you are
       not using a mouse then use <Alt-F10>.
