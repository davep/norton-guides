 Move/Re-size
 Change the size or position of a window.
──────────────────────────────────────────────────────────────────────────────

 This option will allow you to change the size and/or position of the
 currently selected window. If the window that you want to change the size of
 or move, is not the currently selected window, then you will need to change
 to it, otherwise you will change the wrong window.

 Once you have chosen this option, the border of the window will change shape
 and/or colour. You can move the window by using the arrow keys to move the
 window up, down left or right, but not up and down, or left and right at the
 same time.

 The following keys will have some action on the window when it has been
 selected as THE window:

        <PageUp> - moves the window rapidly to the top of the screen
        <PageDn> - moves the window quite quickly to the bottom of the screen
        <Home>   - moves the window like a bat out of hell to the left side
                   of the screen
        <End>    - moves the window like a policeman chasing an Escort
                   Cosworth to the right hand side of the screen

 By using the <Shift> and cursor keys you can change the size of the window.
 To change the size of a window by leaps and bounds you can use the following
 keys:

  <Shift><PageUp> - moves the window rapidly to the top of the screen
  <Shift><PageDn> - moves the window quite quickly to the bottom of the screen
  <Shift><Home>   - moves the window like a bat out of hell to the left side
                    of the screen
  <Shift><End>    - moves the window like a policeman chasing an Escort
                    Cosworth to the right hand side of the screen

 To keep the changes press ◄──┛. To restore the original position
 and size of the window, press <Esc>

 The hot-key combination for this item is <Alt><M>.
