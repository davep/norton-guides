 Application
 Selects the application window as the current window.
──────────────────────────────────────────────────────────────────────────────

 This option will bring up the application screen, if it is not already on
 the screen, and select it as the current window.

 When selected, with the mouse, the right mouse cursor will allow you to
 pop-up the screen ruler and allow you to move it around the application
 window. If you are not using the mouse, then selecting the application
 window with <F4> or <Alt><0> then simply pressing any of the cursor keys
 will select and move the ruler.

 The hot-key combination for this item is <F4>.
