 Again
 Repeats a previous search.
──────────────────────────────────────────────────────────────────────────────

 This option will repeat a previous search. Whatever you had entered in a
 previous search, either Text Search or a Function Search will be repeated.

 If the search proves successful, the source code window will move to the
 line of code that contains the search text

 The hot-key combination for this item is <Shift><Alt><A>.
