 The workarea screen
 An in-depth view of the database and all the tables.
──────────────────────────────────────────────────────────────────────────────

 The workarea screen provides a high amount of detail about each of the
 workareas. You can access the workarea screen by pressing the <F6> key as
 long as there is at least one workarea open.

 Once in this screen you may press <F1> for help.

 The only thing that you need to know about, apart from the <F1> key is that
 when you leave this screen, it does not remember the current workarea and
 return you to this workarea by default the next time that you press <F6>.

 You may change this so that MrDebug does remember the workarea that you last
 moved the cursor to before you came out of this screen. Under the options
 menu is an item called "Save .DBF List Area", you may toggle this item on to
 remember the workarea.
