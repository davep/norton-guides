 Alt-E - Select the View Menu
 This option selects the View pull down menu
──────────────────────────────────────────────────────────────────────────────

 This option selects and displays the View menu.

 The View menu allows you to view, the application screen, the workarea
 screen, the error window, the workarea window, a list of functions in the
 current .PRG, the symbol list (real mode only for the mo), the call stack
 and memory status.
