 Resume
 Restore current line.
──────────────────────────────────────────────────────────────────────────────

 This item will refresh the source code window, the stack window and the
 variable window.

 By opening other files, or changing the current function within the call
 stack window, you have the ability to view different source code .PRGs or
 different functions. The source code window and the variable window will
 change to reflect the new values.

 The 'Resume' option will restore the current place in the call stack window,
 the variable window and the source code window.

 The hot-key combination for this item is <Shift><Alt><R>.
