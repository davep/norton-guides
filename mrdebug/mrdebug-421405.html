 SetKey Definitions
 Shows you a list of keys that have been assigned actions.
──────────────────────────────────────────────────────────────────────────────

 Summoning this option will display a list of keys that have had actions
 assigned to them.

 The list will only contain keys that have been assigned actions up to the
 point in the program when you summon the SetKey() Definitions.

 An example of a SetKey() Definitions window is shown below:

    ┌────────────────────── SetKey() Definitions ────────────────────────┐
    │SetKey(  -3) {|| SHOUT_AT_N() }                                     ↑ 
    │SetKey(  -2) {|| MAIL_SYSTE(GETENV("USER_NAME")) }                  █ 
    │SetKey(  -1) {|| CREATE_GPF() }                                     █ 
    │SetKey(  28) {|| HELP_ME() }                                        ↓ 
    └────────────────────────────────────────────────────────────────────┛ 
                                                                           


 The hot-key combination for this item is <Shift><Alt><K>.
