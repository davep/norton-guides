 Alt-F9 - Pass
 Place a counter on the line under the cursor.
──────────────────────────────────────────────────────────────────────────────

 This option will enable you to keep track of the number of times that a line
 of code is executed. Use the arrow keys to move the cursor to the line that
 you wish to set a pass point on and select this feature.

 A counter will then appear by the side of the line of code in the source
 code window. Each time that the line of code is executed the counter will
 increment.

 This option can be undone by pressing <Alt><F9> on the line that contains
 the pass point, or by deleting it from the pass point list
