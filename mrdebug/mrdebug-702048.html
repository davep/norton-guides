 List disabled Altd()s Points
──────────────────────────────────────────────────────────────────────────────

 This option will display a list of all the disabled Altd()s that you have
 set.

 The list will contain the name of the source code file and the line number
 that the disabled Altd()s has been set on for each ALTD() function that you
 have within your code.

    █▀▀ Disabled Altd() List ▀▀█     The ↑ or ↓ keys will allow you to choose
    █ TEST.PRG              41 █     a disabled Altd(), press ◄──┛ on the
    █ TEST.PRG              90 █     disabled Altd()s point and the source
    █▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄█     code window will change to enable
                                     you to view the disabled Altd()s point.

 To restore the source code window, press <Shift><Alt><R>.

 This option will remain dimmed if there are no disabled Altd()s points set
 and you will not be able to select it.

 The Command window short-cut command for this is LIST ALTD.
