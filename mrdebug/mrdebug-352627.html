 The workarea window
 A brief summary of the database
──────────────────────────────────────────────────────────────────────────────

 The Workarea window allows you to have a view of the current workarea
 contained within a window on the screen at all times during your program
 execution. You may toggle the display of this window on or off with the
 <Alt><F6> key combination.

 An example of which is shown below:

  ┌─ 6 Workarea 1 Status ───────────────┐
  │  WorkArea:1 - MICHELLE              │
  │    Record:333/1795                  │
  │     Index:1 - E:\RCV\MICHELLE.NTX   │
  │ Index Key:FILM+YEAR+RATING          │
  │Actual Key:"ZZZ9991983Hmmmmm....Yes" │
  └─────────────────────────────────────┛

 By default only the above information is shown. However if you expand the
 size of the window, you will be able to see the field information:

  ┌─ 6 Workarea 1 Status ───────────────┐
  │  WorkArea:1 - MICHELLE              │
  │    Record:333/1795                  │
  │     Index:1 - E:\RCV\MICHELLE.NTX   │
  │ Index Key:FILM+YEAR+RATING          │
  │Actual Key:"ZZZ9991983Hmmmmm....Yes" │
  │─────────────────────────────────────│
  │[ 1] DATEINPUT  : 23/12/93          ↑│
  │[ 2] FILM       : ZZZ999            █│
  │[ 3] YEAR       : 1983              ▒│
  │[ 4] REFERENCE  : 987123645         ▒│
  │[ 5] STARS      : 00000001          ▒│
  │[ 6] PARTNUMBER : 010               ▒│
  │[ 7] RATING     : Hmmmmm....Yes     ↓│
  └─────────────────────────────────────┛
