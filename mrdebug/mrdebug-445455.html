 Locals
 Toggle the display of local variables on or off.
──────────────────────────────────────────────────────────────────────────────

 This option toggles the displaying of the local variables wtihin the
 variables window.. If the variables window is nnot open, then the variables
 window will automatially open.

 When you choose this option the LOCAL variables within the current
 function/procedure will be shown in the variables window.

 The hot-key combination for this item is <Alt><L>.
