 Variable Casing
 Toggles the case of variables within the variable window.
──────────────────────────────────────────────────────────────────────────────

 This option, when turned on, will apply the following to variable names:

        If the variable name starts with the same letter as the variable
        type, then the variable name will converted to lower case and the
        second letter of the variable name will be converted to upper case.

        If a variable name begins with an 'x' letter, is will also be
        converted to lower case and have it's second letter changed to upper
        case

        Otherwise the variable name will be displayed using UPPER CASE
        letters.

 When turned off, all variable names will be displayed in Upper Case.
