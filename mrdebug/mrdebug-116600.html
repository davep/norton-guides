 The Profiler Window                                                   <Alt-9>
 About The Profiler Window.
──────────────────────────────────────────────────────────────────────────────

 The Profiler window contains a list of the functions that have been
 called by your program and various columns that display information for each
 function such as the slowest call of that function, the fastest call of the
 function, the number of times that the function has been called and the
 average (mean) time of all of the calls to that function.

  ┌─ 9 Profile (1/8) [8.1223] ─────────────────────────────────────────────┐
  │• Filename-- Function----- -----Calls -------Total Time ----------Averag↑
  │  RCV.PRG    GETSET_SET             1            0.0073            0.007█
  │  RCV.PRG    SAVECURTAI             1            0.0758            0.075▒
  │  RCV.PRG    INIT                   1            2.9058            2.905↓
  └←████████████████████████████████████████████▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒→┛


 The Profiler only works on .OBJ files that have been compiled with the /b
 switch.

 The profiler has to be enabled for the profiling to take place.

 The following is an example of the pop-up menu for the Profiler window,
 which may be summoned either by clicking the right mouse button on the
 window or by using <Alt-F10>.
