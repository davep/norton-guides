 Trace
 Set a trace point.
──────────────────────────────────────────────────────────────────────────────

 This allows you to trace a variable or an expression through a piece of
 code. Each time that the expression or variable changed during the lifetime
 of the tracepoint, the tracepoint changes and the program stops and MrDebug
 takes over.

 The trace point will have the new value and the old value will be displayed
 along side it in the watch window.

 The command window will also display a message similar to the following:

   -->Trace Point Break : LCONTINUE (Now:NIL, was:.T.)

 This tells us that the variable LCONTINUE has changed from being a logical
 value of .T. to NIL - sure helped me to track down a problem with my code!

 Please Note: If Fast Run Mode is enabled, this option will not work.

 The hot-key combination for this item is <Alt><T>.
