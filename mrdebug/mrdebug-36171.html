 Alt-M - Move/Re-size 
 Select move/re-size mode.
──────────────────────────────────────────────────────────────────────────────

 This will allow you to re-size and/or move the current window. This option
 will change the border of the current window (so that you know which window
 you will be playing with). You may then use the arrow keys to move the
 window. Pressing the Escape key, abort the move and re-positions the window
 in it's original place. Pressing ◄──┛ accepts the new position for the
 window.

 You may use the <Shift> and cursor keys to change the size of the window.
 Pressing <Esc> aborts the re-size, ◄──┛ accepts the new size of the window.
