 Workarea List
 Colour settings for the Workarea List.
──────────────────────────────────────────────────────────────────────────────

 The following colour settings allows you to modify the colour settings for
 the Workarea list:

 Alias Text - this is the colour used to display standard alias names
 within the alias list.

 Alias Highlight Item - this is the colour used to display the alias that
 has a highlight bar on it.

 Alias Default Item - this is the colour used to show the default, or
 current, or selected alias of the program.

 Alias Default Highlight Item - this is the colour setting to use when
 the highlight bar is on the currently selected workarea.

 Alias Border Unselected - this is the colour setting that is used when
 the Alias list is not the currently selected box.

 Alias Border Selected - this is the colour setting used to display the
 alias box border when it is the currently selected box.

 Index Text - this is the colour to use to show standard text when
 displaying index entries within the index box.

 Index Highlight Item - this is the colour setting to use when displaying
 an index entry with the highlight bar on it.

 Index Default Item - this is the colour setting to use when displaying
 the currently selected index order.

 Index Default Highlight Item - this is the colour setting used to
 display the highlight bar on the currently selected index order.

 Index Border Unselected - this is the colours to use when displaying the
 border for the index box, if it is no the currently selected box.

 Index Border Selected - this is the colour setting to use to display the
 index border if it is the currently selected box.

 Field Text - this is the colour setting that will be used to show all
 non-highlighted text.

 Field Highlight Item - this is the colour setting that is used to show a
 highlighted item in the field list.

 Field Deleted Record - this is the colour item to show allow deleted
 records to stand out.

 Field Deleted Highlight Record/Item - this is the colour setting to use
 to display the currently highlighted item if it is also marked for deletion.

 Field Border Unselected - this is the colours to use when displaying the
 border for the fields box, if it is no the currently selected box.

 Field Border Selected - this is the colour setting to use to display the
 fields border if it is the currently selected box.

 Status Text - this is the colour setting that is used to  display the
 standard text such as titles.

 Status Information - this is the colour setting that is used to display
 information such as the record number, file status flags etc.

 Status Border Unselected - this is the colour setting that is used to
 display the border when the status box is not the currently selected box.

 Status Border Selected - this is the colour setting that is used to
 display the border when the Status box is selected.

 Locks Text - this is the colour setting used to display standard text
 items, such as the record numbers that are locked, within the Locks window.

 Locks Highlight - this is the colour setting that is used to display the
 highlight bar on a record lock number within the list.

 Locks Border Unselected - this is the colour setting that is used to
 display the border when the locks box is not the currently selected box.

 Locks Border Selected - this is the colour setting that is used to
 display the border when the Status box is selected.
