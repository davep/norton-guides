 List Boxes
 Allows you to find out where your points are.
──────────────────────────────────────────────────────────────────────────────

 The List option from the options menu allows you to view all of the places
 where you have set various items up, such as Break Points, Trace points,
 Watch Points, Pass Points and Disabled AltD() functions.

 You simply pick which one you want and the list box appears with a list of
 each of the points that you have set with the line number and the name of
 the .PRG in which it appears. You may use the <↑> or <↓> to scroll up and
 down the list, if they cannot all be displayed at once.

 Should you press ◄──┛ on one of the items in the list, then the source code
 window will move to that line of code, changing source code .PRGs if
 necessary.

 Even individual break points or trace points can be disabled.
