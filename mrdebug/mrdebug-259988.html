 I've added asserts to my code but they don't work. I've included MrDebug.Ch,
 added ASSERT and TRACE commands throughout my source code but I can't seem
 to get anything to happen - nothing appears in the Assert/Trace window.

 Check to see that you have compiled each of the .PRG's that you wish to use
 MrDebug's assertion checking in with a /dMRDEBUG - this compiler switch
 tells Clipper to include the Assertion checking code at compile time in the
 module.

 Finally check to see that you haven't disabled Assertion checking. The
 option to disable Assertion checking is held on the Assert/Trace sub-menu of
 the Options menu. If there is a cross (x) on the menu item labelled "Disable
 Asserts/Traces" then Assertion checking has been disabled. You may toggle
 this off by pressing <SpaceBar> when the highlight bar is on this menu item.
