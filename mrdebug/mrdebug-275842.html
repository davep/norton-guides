 What are Fixup overflow errors - what can I do about them?
──────────────────────────────────────────────────────────────────────────────

 The most likely cause of this problem is that you have linked in a non
 CA-Clipper .OBJ file as the first object module within your source code.

 Check to make sure that the first .OBJ is a CA-Clipper .OBJ.

 Also check to see that MrDebug is not the first file referenced within your
 link script. MrDebug must be included in the link script after at least one
 CA-Clipper .OBJ entry and before any references to CLIPPER.LIB
