 Screen Swap  (default is on)                            (80x25 mode users)
 Toggle the screen swapping option on or off.
──────────────────────────────────────────────────────────────────────────────

 If you are running MrDebug in 80 columns by 25 rows, then you'll probably
 want to turn this option off so that the debugger does not keep switching
 back to the application screen each time a line of code is executed.

 NOTE : For users running in text modes that support 132 columns, this option
        has little or no affect as the application screen is always
        displayed.
