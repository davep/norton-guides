 MRD is ignoring an AltD() in my code.
 You have probably disabled all ALTD() functions
──────────────────────────────────────────────────────────────────────────────

 MrDebug has the ability to ignore ALTD() functions that have you have
 hard coded into your source code and you have probably disabled either an
 ALTD() or all ALTD()'s within your code.

 To delete a disabled ALTD():

        1. Bring the Point menu by pressing <Alt><I>
        2. Bring up the List menu by typing <L>
        3. List the disabled ALTD()'s by pressing <D>
        4. Delete the items that appear in the list with the <DEL> key.
