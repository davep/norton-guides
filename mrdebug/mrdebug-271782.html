 How do I link with CA-Clipper 5.3?
 Getting MrDebug to work with CA-Clipper 5.3
──────────────────────────────────────────────────────────────────────────────

 To enable MrDebug to work with CA-Clipper 5.3, you need to include the
 MrDebug extensions for CA-Clipper 5.3. To do this, simply add the line FI
 MRDBG53.LIB to your link script, followed by an entry for MrDebug.LIB using
 the LIB command.

 For example:

        FI test
        FI utils
        FI Mrdbg53.lib

        LIB MrDebug

 Remember that if you include Mrdbg53.lib using the FILE command, you must
 include the .LIB extension for the file - otherwise it won't work.
