 Shift-Alt-P - PPO Mode (V-Split)
 Toggles display of PPO file.
──────────────────────────────────────────────────────────────────────────────

 This command will allow you to toggle the display of the .PPO window. The
 .PPO window may be moved and re-size should you so wish to do so.

 The source code module must be compiled with the /p switch, otherwise the
 .PPO window will be empty. You may move and re-size this window so that it
 appears below the source code window, showing only the current line of
 code, or even place it along side the source code window - ideal in 132
 columns by 60 rows text mode.
