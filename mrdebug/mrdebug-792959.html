 Popup Windows
 Colour settings for any Pop-Up Windows.
──────────────────────────────────────────────────────────────────────────────

 The following colour settings allows you to modify the colour settings for
 any Pop-Up Windows:

 Text - this is the colours setting used to display any standard text
 that is displayed within a pop-up window.

 List Item Text - this is the colour used to show actual entries that
 appear in any lists that appear, for example this colours is used to show
 the value of the SETs within the set window.

 List Item Name - this is the colours used to show the names of any items
 that appear in lists, for example this item is used to show the names of the
 SETs within the Sets list.

 List Item Text, Highlighted - this is the colour used to show the
 highlighted actual entry that appears in any lists, for example this colour
 is used to show the highlighted value of the SET within the set window.

 List Item Name, Highlighted - this is the colour used to show the
 highlighted name of an item that appears in any lists, for example this item
 is used to show the highlighted name of the SETs within the Sets list.

 Scroll Bar  - the colours used for any scroll bars that may be attached
 to a pop-up window.

 Border Unselected - this is the colour that is used to show the border
 for any pop-up windows within a pop-up window that is not the currently
 selected pop-up window.

 Border Selected - this is the colour used to show the border for any
 pop-up window.
